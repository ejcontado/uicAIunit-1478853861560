Footer bordered with links

To create a footer with links, firstly, contents that are non-footer are wrapped in n-footer-wrapper. Place contents of the footer inside n-footer or n-footer-bordered. Place footer links in the footer inside n-footer-left whose links have the class name n-footer-links. Copyright is placed inside n-footer-right as normal. 


    <div class="n-footer-wrapper">
        <!-- page items -->
    </div>

    <!--Bordered footer with links -->
    <div class="n-footer-bordered">
        <div class="n-footer-left">
            <ul>
                <li><a href="#" class="n-footer-links">Link 1</a></li>
                <li><a href="#" class="n-footer-links">Link 2</a></li>
                <li><a href="#" class="n-footer-links">Link 3</a></li>
            </ul>
        </div>
        <div class="n-footer-right">
            <div class="n-footer-copyright">
                ©2016 Nokia. All rights reserved.
            </div>
        </div>
    </div>

JS & CSS dependency:

For footer components, it will be ok to import following css file:
<link href="../wulfdist/css/wulf.basic.min.css" rel="stylesheet"/>
<link href="../wulfdist/css/wulf.footer.min.css" rel="stylesheet"/>
