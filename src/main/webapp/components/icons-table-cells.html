Icons in table cells

Icons in table cells

Description Icons
Icon status
Icon control

JS & CSS dependency:

For icons in table cells components, it will be ok to import following css file :
<link href="../wulfdist/css/wulf.basic.min.css" rel="stylesheet"/>

Status icons and clickable icons in table cells have an added background to provide the necessary color contrast in a selected table row or cell.

Add n-cell-icon on a div to apply the style.
Use the n-cell-icon-container for the default alignment or you can specify your own alignment inside cells. 


    <!-- Icon Status -->
    <td>
        <div class="n-cell-icon-container">
            <div class="n-cell-icon">
                <span class="icon icon-fault-critical"></span>
            </div>
        </div>
    <td>

Use n-cell-icon-control for icon control. 


    <!-- Icon Control -->
    <div class="n-cell-icon-container">
        <button class="n-cell-icon-control">
            <span class="icon icon-fault-critical"></span>
        </button>
    </div>
