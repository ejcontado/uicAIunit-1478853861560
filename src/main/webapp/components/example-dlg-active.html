JS & CSS dependency:

For active dialog components, it will be ok to import following css file:
<link href="../wulfdist/css/wulf.basic.min.css" rel="stylesheet"/>
<link href="../wulfdist/css/wulf.buttons.min.css" rel="stylesheet"/>
<link href="../wulfdist/css/wulf.dialog.min.css" rel="stylesheet"/>

To use the dialogs, see the examples below. 


<!-- Standard dialog -->
<a href="#ActiveDialog" data-backdrop="false" data-toggle="modal">Active dialog</a>

You can make use of the active dialog by add class active.See the examples below. 


<!-- Standard dialog -->
<div class="modal active" id="ActiveDialog" role="dialog" aria-labelledby="myModalDlgLabel-active" aria-hidden="true" tabindex='-1'>
<div class="modal-dialog">
            <div class="modal-content">
                <div class="modal-header">
                    <div class="n-dialog-header-fg-filler"><div>
                    <div class="n-dialog-header-bg-filler"></div>
                    
                    <div class="n-dialog-header-curve">
                        <div class="fg-mask"></div>
                        <div class="bg-mask"></div>
                        <div class="fg-corner"></div>
                        <div class="bg-corner"></div>
                    </div>
                    <i role="button" class="icon icon-close-rounded" data-dismiss="modal" aria-label="Close dialog"></i>
                    <h1 class="modal-title-active">Active Dialog</h1>
                </div>
                <div class="modal-body">
                    Text
                </div>
                <div class="modal-footer">
                    <button class="btn btn-standard" data-dismiss="modal">OK</button>
                    <button class="btn btn-standard" data-dismiss="modal">Cancel</button>
               </div>
            </div>
        </div>
</div>
