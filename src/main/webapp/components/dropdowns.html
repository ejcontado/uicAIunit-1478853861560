JS & CSS dependency:

for Pull down components,you can use following css import code:
<link href="../wulfdist/css/wulf.basic.min.css" rel="stylesheet"/>
<link href="../wulfdist/css/wulf.buttons.min.css" rel="stylesheet">
<link href="../wulfdist/css/wulf.dropdowns.min.css" rel="stylesheet">
<link href="../wulfdist/css/wulf.tooltip.min.css" rel="stylesheet">

For getting the disabled version, add disabled as a class into the top div. For getting the dark version, add selectlist-dark as a class into the top div. For getting the small version, add selectlist-small as a class into the top div. For more information about this component, see FuelUX documentation about Selectlist.


    <!-- Basic Pull down -->
    <div class="btn-group selectlist selectlist-resize" data-resize="none" data-initialize="selectlist" id="mySelectlist">
    <button class="btn btn-default dropdown-toggle" data-toggle="dropdown" type="button">
    <span class="selected-label"></span>
    <span class="selected-caret" ><span class="caret"></span></span>
    <span class="sr-only">Toggle Dropdown</span>
    </button>
    <ul class="dropdown-menu" role="menu">
    <li data-value="1"><a href="#"><span>One</span></a></li>
    <li data-value="2"><a href="#"><span>Two</span></a></li>
    </ul>
    <input class="hidden hidden-field" name="mySelectlist" readonly="readonly" aria-hidden="true" type="text"/>
    </div>

    <!-- JS-script needed -->
    <script">
        require(['jquery', 'wulf/dropdowns'], function($){ });
    </script>
