Footer

To use a footer, contents that are non-footer is wrapped in n-footer-wrapper. Then contents of the footer are placed inside n-footer or n-footer-bordered. See the subpages for detailed examples. 


    <!-- Page content wrapper -->
    <div class="n-footer-wrapper">
        <!-- page items -->
    </div>

JS & CSS dependency:

For footer components, it will be ok to import following css file:
<link href="../wulfdist/css/wulf.basic.min.css" rel="stylesheet"/>
<link href="../wulfdist/css/wulf.footer.min.css" rel="stylesheet"/>
