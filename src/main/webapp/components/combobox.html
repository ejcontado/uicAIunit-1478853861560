JS & CSS dependency:

For Combo box components, it will be ok to import following css file :
<link href="../wulfdist/css/wulf.basic.min.css" rel="stylesheet"/>
<link href="../wulfdist/css/wulf.buttons.min.css" rel="stylesheet"/>
<link href="../wulfdist/css/wulf.combobox.min.css" rel="stylesheet"/>
<link href="../wulfdist/css/wulf.dropdowns.min.css" rel="stylesheet"/>
<link href="../wulfdist/css/wulf.inputfield.min.css" rel="stylesheet"/>
<link href="../wulfdist/css/wulf.tooltip.min.css" rel="stylesheet"/>

For getting the disable combo textfield, add disabled at the end of both input and button elements.

For more information about this component, see FuelUX documentation about Combobox.

If want to pre-select something, add class selected to that particular li item.


    <!-- Combobox (textfield) -->
    <label class="control-label">Standard:</label>
    <div class="input-group input-append dropdown combobox" data-initialize="combobox" id="myCombobox">
        <input type="text" class="form-control n-inputfield">
        <div class="input-group-btn">
            <button type="button" class="btn btn-default dropdown-toggle" data-toggle="dropdown"><span class="caret"></span></button>
            <ul class="dropdown-menu dropdown-menu-right">
                <li data-value="1"><a href="#"><span>One</span></a></li>
                <li data-value="2"><a href="#"><span>Two</span></a></li>
            </ul>
        </div>
    </div>

    <!-- Disabled combobox (textfield)-->
    <label class="control-label label-disabled">Disabled:</label>
    <div class="input-group input-append dropdown combobox" data-initialize="combobox" id="myCombobox">
        <input type="text" class="form-control" disabled>
        <div class="input-group-btn">
            <button type="button" class="btn btn-default dropdown-toggle" data-toggle="dropdown" disabled><span class="caret"></span></button>
            <ul class="dropdown-menu dropdown-menu-right">
                <li data-value="1"><a href="#"><span>One</span></a></li>
            </ul>
        </div>
    </div>

    <!-- With disabled items combobox (textfield) -->
    <label class="control-label">With disabled items:</label>
    <div class="input-group input-append dropdown combobox" data-initialize="combobox" id="myCombobox">
        <input type="text" class="form-control n-inputfield">
        <div class="input-group-btn">
            <button type="button" class="btn btn-default dropdown-toggle" data-toggle="dropdown"><span class="caret"></span></button>
            <ul class="dropdown-menu dropdown-menu-right">
                <li data-value="1"><a href="#"><span>One</span></a></li>
                <li data-value="2" class="disabled" ><a href="#"><span>Two</span></a></li>
            </ul>
        </div>
    </div>

    <!-- Clear button combobox (textfield)-->
    <div class="form-group">
        <label class="col-sm-5 control-label" style="width: 180px">Clear button:</label>
        <div class="col-sm-8">
            <div class="input-group input-append dropdown combobox n-cancel-button" data-initialize="combobox">
                <input type="text" class="form-control n-inputfield" placeholder="Text">
                <a class="n-clear-button-icon hide-element" href="javascript:void(0)">
                    <span role="button" aria-label="clear content" class="icon icon-close"></span>
                </a>
                <div class="input-group-btn">
                    <button type="button" class="btn btn-default dropdown-toggle" data-toggle="dropdown">
                        <span class="caret"></span>
                    </button>
                    <ul class="dropdown-menu dropdown-menu-right" role="menu">
                        <li data-value="1"><a href="#"><span>One</span></a></li>
                        <li data-value="2"><a href="#"><span>Two</span></a></li>
                        <li data-value="3"><a href="#"><span>Three</span></a></li>
                    </ul>
                </div>
            </div>
        </div>
    </div>

    <!-- JS-script needed -->
    <script src="./wulfdist/js/dependencies/fuelux.js" type="text/javascript"></script>
    <script>
        $('#myCombobox').combobox();
        $(document).ready(function () {
            $('ul.dropdown-menu.dropdown-menu-right li').click(function () {
                $('ul.dropdown-menu.dropdown-menu-right li').removeClass('selected');
                $(this).addClass('selected');
            });
        });
    </script>

For getting the small combo button version, add n-combo-btn-small as a class into the top div and for the label use control-label-sm. For getting the disabled combo button, add disabled at the end of button element and for label add label-disabled. For getting the contextual menu combo button, use contextual-menu-label instead of selected-label. If want to pre-select something, add class selected to that particular li item.

    <label class="control-label">Standard:</label>
