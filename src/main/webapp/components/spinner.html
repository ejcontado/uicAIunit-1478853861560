Spinner



JS & CSS dependency:

For spinner components, it will be ok to import following css file:
<link href="../wulfdist/css/wulf.basic.min.css" rel="stylesheet"/>
<link href="../wulfdist/css/wulf.inputfield.min.css" rel="stylesheet"/>
<link href="../wulfdist/css/wulf.spinner.min.css" rel="stylesheet">

For getting the disabled version, add disabled as a class into the top div, <input> tag, <label> tag and <button> tag. Also for the label <label for="mySpinnerDisabled" class="control-label label-disabled">Disabled spinner:</label>
For more information about this component, see FuelUX documentation about spinbox. 


    <!-- Basic spinner -->
    <div class="spinner-container">
    <label for="mySpinner" class="control-label">Normal spinner:</label>
    <div class="spinbox" data-initialize="spinbox" id="mySpinner">
    <input id="s-normal" type="text" class="form-control spinbox-input n-inputfield">
    <div class="spinbox-buttons btn-group btn-group-vertical">
    <button type="button" class="btn btn-default spinbox-up btn-xs">
    <span class="icon icon-arrow-up"></span>
    <span class="sr-only">Increase</span>
    </button>
    <button type="button" class="btn btn-default spinbox-down btn-xs">
    <span class="icon icon-arrow"></span>
    <span class="sr-only">Decrease</span>
    </button>
    </div>
    </div>
    </div>

    <!-- small spinner -->
    <div class="spinner-container">
    <label for="mySpinner" class="control-label">Normal spinner:</label>
    <div class="spinbox n-spinbox-small" data-initialize="spinbox" id="mySpinner">
    <input id="s-normal" type="text" class="form-control spinbox-input n-inputfield">
    <div class="spinbox-buttons btn-group btn-group-vertical">
    <button type="button" class="btn btn-default spinbox-up btn-xs">
    <span class="icon icon-arrow-up"></span>
    <span class="sr-only">Increase</span>
    </button>
    <button type="button" class="btn btn-default spinbox-down btn-xs">
    <span class="icon icon-arrow"></span>
    <span class="sr-only">Decrease</span>
    </button>
    </div>
    </div>
    </div>

    <!-- JS-script needed -->
    <script src="../wulfdist/js/dependencies/fuelux.js" type="text/javascript"></script>
    <script>
    $('#mySpinner').spinbox();
    $('#mySpinnerDisabled').spinbox();
    $('#mySmallSpinner').spinbox();
    $('#mySmallSpinnerDisabled').spinbox();

    //for disabled version
    $('#mySpinnerDisabled').off("mousedown");
    $('#mySmallSpinnerDisabled').off("mousedown");
    </script>

Modify the width of spinner input in CSS:


    .spinbox .spinbox-input {
    width: 200px;
    }
