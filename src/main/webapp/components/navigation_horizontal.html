This secondary navigation is an alternative to tab navigation. Unlike tab navigation, secondary navigation acquires a drop shadow when the page scrolls upwards.
To use the this, add a row with an n-banner-secondary-row class together with an unordered list with the classnav-secondary-horizontal: 


    <nav class="n-banner">
        ......
        <div class="row n-banner-secondary-row">
            <ul class="nav nav-secondary-horizontal">
                <li>
                    <a href="#">
                        <span>Item One</span>
                    </a>
                </li>
                //More list items
            </ul>
        </div>
    </nav>

The condensed product banner can be employed upon scroll in place of the standard banner.
To enable, just add some javascript like this on your page to automatically add the class condensed on the banner upon scroll. 


    <script>
        $(document).on('scroll', function() {
            var nBanner = $('.nav-secondary-horizontal').parents('nav.n-banner');

            if($(this).scrollTop()) {
                nBanner.addClass('condensed');
            } else {
                nBanner.removeClass('condensed')
            }
        });
    </script>
