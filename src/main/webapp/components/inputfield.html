Text field

Standard text fields

JS & CSS dependency:

For text field components, it will be ok to import following css file:
<link href="../wulfdist/css/wulf.basic.min.css" rel="stylesheet"/>
<link href="../wulfdist/css/wulf.inputfield.min.css" rel="stylesheet">

Restrictions: Do not use placeholder in disabled text field. 


    <!-- Standard size text field -->
    <label for="Example1" class="control-label">Normal field:</label>
    <input id="Example1" type="text" class="form-control n-inputfield" placeholder="Label inside">

    <!-- Disabled text field -->
    <label for="Example2" class="control-label label-disabled">Disabled field:</label>
    <input id="Example2" type="text" class="form-control n-inputfield" disabled value="Disabled field">

    <!-- Standard size not-editable text field, e.g. no edit rights -->
    <input class="form-control n-inputfield n-inputfield-uneditable" readonly>

    <!-- Static text -->
    <input class="form-control n-form-control-static" readonly>

    <!-- Clearable text field -->
    <div class="n-inputfield-clearable">
        <input type="text" class="form-control n-inputfield">
        <a class="n-inputfield-control-icon hide-element">
            <span role="button" aria-label="clear textfield content" class="icon icon-close"></span>
        </a>
    </div>

    Note: You may initially hide the n-input-control-icon to display it only when there is text inside:
    .hide-element{
        display: none;
    }

Text field with Mandatory icon() has functionality. Others with indicators will support functionality in future.


    <!-- Basic pattern in the form -->
    <div class="form-group has-feedback">
        <div>
            <input type="text" class="form-control n-inputfield">
            <a class="form-control-feedback"><span class="icon icon-input-error"></span></a>
        </div>
    </div>

    <!-- Required field --> class="icon icon-mandatory" feedback type class="input-required"
    <!-- Invalid field --> class="icon icon-input-error" feedback type class="input-error"
    <!-- Changed field --> class="icon icon-edited" feedback type class="input-changed"
    <!-- Valid field --> class="icon icon-ok" feedback type class="input-valid"

    Feedback type can be used with form-group class or with form-control-feedback class

    <!-- Has an error -->
    <input type="text" class="form-control n-inputfield has-error">
