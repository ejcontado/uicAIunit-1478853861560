JS & CSS dependency:

For login components, it will be ok to import following css file:
<link href="../wulfdist/css/wulf.basic.min.css"/>
<link href="../wulfdist/css/wulf.buttons.min.css" rel="stylesheet"/>
<link href="../wulfdist/css/wulf.inputfield.min.css" rel="stylesheet"/>
<link href="../wulfdist/css/wulf.login.min.css" rel="stylesheet"/>
<link href="../wulfdist/css/wulf.links.min.css" rel="stylesheet"/>
<link href="../wulfdist/css/wulf.panel.min.css" rel="stylesheet"/>
<link href="../assets/css/demo.css" rel="stylesheet"/>

For login error component, it will be ok to extra import following this another css file:
<link href="../wulfdist/css/wulf.alert.min.css" rel="stylesheet"/>
