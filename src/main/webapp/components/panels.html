JS & CSS dependency: 
For Panel components, it will be ok to import following css file:
<link href="../wulfdist/css/wulf.basic.min.css" rel="stylesheet"/>
<link href="../wulfdist/css/wulf.panel.min.css" rel="stylesheet">

You can create a basic panel container by creating a div container with class panel. The panel consists of the panel container, a container for the panel header, as well as a container for the panel content, each with their own class: 


  <div class="panel panel-shadow panel-blue-cap">
  <div class="panel-heading">
    <h1>Available Operations</h1>
  </div>
  <div class="panel-body">
    ...
  </div>
  </div>

The panel component also provides a couple of predefined classes for changing the visual appearance of the panel. You can use panel-shadow class to add a drop shadow, as well as panel-blue-cap class to add a blue cap on top of the panel.

You can also use predefined sections and single rows to split your panel into distinct parts: 


  <div class="panel-body">

  <div class="panel-section">
    <h2>Title of section</h2>
    ...
  </div>

  <div class="panel-single-row">
    <h2>Title or label</h2>
  </div>

  <div class="panel-section">
    <h2>Title of section</h2>
    ...
  </div>
  </div>
