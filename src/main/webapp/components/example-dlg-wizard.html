JS & CSS dependency:

For dialog wizard components, it will be ok to import following css file :
<link href="../wulfdist/css/wulf.basic.min.css" rel="stylesheet"/>
<link href="../wulfdist/css/wulf.dialog.min.css" rel="stylesheet"/>
<link href="../wulfdist/css/wulf.tabpane.min.css" rel="stylesheet"/>

Wizard dialogues always contain a dynamic progress bar in the header, clearly showing the number of steps in the process, and buttons at the base to navigate through those steps. To use the wizard dialog, see the examples below.

<!-- Dialog wizard -->
<div class="modal n-dlg-wizard" id="myWizardDialog" role="dialog" aria-labelledby="myModalDlgLabel-subhead" aria-hidden="true" tabindex='-1'>
    <div class="modal-dialog">
        <div class="modal-content">
            <div class="modal-header">
                    <div class="n-dialog-header-fg-filler"></div>
                    <div class="n-dialog-header-bg-filler"></div>
                    <div class="n-dialog-header-curve">
                        <div class="fg-mask"></div>
                        <div class="bg-mask"></div>
                        <div class="fg-corner"></div>
                        <div class="bg-corner"></div>
                    </div>
                    <i role="button" class="icon icon-close-rounded" data-dismiss="modal" aria-label="Close wizard"></i>
                    <h1 class="modal-title" id="myModalLabel">Create Task</h1>
                    <h2 class="modal-title-subheading" id="myModalLabelVersion">General data</h2>
                </div>
            <div class="modal-body">
                <div class="navbar">
                <div class="navbar-inner">
                    <ul>
                        <li><a href="#tab1" data-toggle="tab"><span>1</span></a></li>
                        ...
                    </ul>
                </div>
            </div>
            <div class="tab-content">
                <div class="tab-pane" id="tab1">
                    <div class="n-dlg-wizard-step-description">Step 1 description</div>
                    <div class="n-dlg-wizard-step-content">It is task 1.</div>
                </div>
                ...
            </div>
            <div class="modal-footer">
                <input type='button' class='btn btn-standard button-previous' name='previous' value='Back'/>
                <input type='button' class='btn btn-standard button-next' name='next' value='Next'/>
                <input type='button' class='btn btn-standard button-last' name='last' value='Finish'/>
                <input type='button' class='btn btn-standard button-first ' name='first' value='Cancel'/>
            </div>
        </div>
    </div>
</div>
