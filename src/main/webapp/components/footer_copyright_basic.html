Footer with copyright

To create a copyright footer, firstly, contents that are non-footer are wrapped in n-footer-wrapper. Place contents of the footer inside n-footer or n-footer-bordered. Then add copyright by placing it inside n-footer-right as seen in the code sample below which uses a non-bordered footer. 


    <!-- Page content wrapper -->
    <div class="n-footer-wrapper">
        <!-- page items -->
    </div>

    <!-- Basic footer -->
    <div class="n-footer">
        <div class="n-footer-right">
            <div class="n-footer-copyright">
                © 2016 Nokia. All rights reserved.
            </div>
        </div>
    </div>
If want border line for the footer, add n-footer-bordered around the footer. 

    <!-- Bordered footer -->
    <div class="n-footer-bordered">
        <div class="n-footer-right">
            <div class="n-footer-copyright">
                © 2016 Nokia. All rights reserved.
            </div>
        </div>
    </div>

JS & CSS dependency:

For footer components, it will be ok to import following css file:
<link href="../wulfdist/css/wulf.basic.min.css" rel="stylesheet"/>
<link href="../wulfdist/css/wulf.footer.min.css" rel="stylesheet"/>
