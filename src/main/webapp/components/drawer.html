Drawer

JS & CSS dependency:

For drawer components, it will be ok to import following css file :
<link href="../wulfdist/css/wulf.basic.min.css" rel="stylesheet"/>
<link href="../wulfdist/css/wulf.flyout.min.css" rel="stylesheet"/>

Drawer is a flyout menu. So we use flyout markup data-markup="flyout". It opens from the bottom of screen. So we set the direction as top (expanse to top) data-direction="top".

We can integrate it with tab component. The tab should point to the content with id: <a href="#content"> and <div id="content">. 


    <div class="n-flyout n-drawer" data-markup="flyout" data-direction="top" data-expand="false">
        <ul class="n-flyout-open n-drawer-tabs">
            <li>
                <a href="#one"><span class="topleft"></span><span class="border-container">Tab one</span></a>
            </li>
            <li>
                <a href="#two"><span class="topleft"></span><span class="border-container">Tab Two</span></a>
            </li>
            <li>
                <a href="#three"><span class="topleft"></span><span class="border-container">Tab Three</span></a>
            </li>
            <li>
                <a href="#four"><span class="topleft"></span><span class="border-container">Tab Four</span><span class="topright"></span></a>
            </li>
        </ul>
        <div id="one" class="n-flyout-container">
            one
        </div>
        <div id="two" class="n-flyout-container">
            Tab two content
        </div>
        <div id="three" class="n-flyout-container">
            Tab three content
        </div>
        <div id="four" class="n-flyout-container">
            Tab four content
        </div>
        <div class="drawer-toggle-up"></div>
        <div class="drawer-shadow"></div>
    </div>
