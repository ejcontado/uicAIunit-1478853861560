Consumer Footer

JS & CSS dependency:

For footer components, it will be ok to import following css file:
<link href="../wulfdist/css/wulf.basic.min.css" rel="stylesheet"/>
<link href="../wulfdist/css/wulf.footer.min.css" rel="stylesheet"/>

To create a consumer footer, firstly, contents that are non-footer are wrapped in n-footer-wrapper. Place contents of the footer inside n-footer or n-footer-bordered. Place footer links in the footer inside n-footer-left. Add copyright inside n-footer-right as normal. Then add n-footer-icons inside n-footer-right and add images wrapped with item-icon inside n-footer-icons as seen in the code sample below. 


    <!-- Page content wrapper -->
    <div class="n-footer-wrapper">
        <!-- page items -->
    </div>

    <!-- Basic footer -->
    <div class="n-footer">
        <div class="n-footer-left">
            <ul>
                <li><a href="#" class="n-footer-links">Link 1</a></li>
                <li><a href="#" class="n-footer-links">Link 2</a></li>
                <li><a href="#" class="n-footer-links">Link 3</a></li>
            </ul>
        </div>
        <div class="n-footer-right">
            <div class="n-footer-copyright">
                © 2016 Nokia. All rights reserved.
            </div>
            <div class="n-footer-icons">
                <a class="item-icon">
                    <img src="../assets/images/wulf.png">
                </a>
                <a class="item-icon">
                    <img src="../assets/images/wulf.png">
                </a>
                <a class="item-icon">
                    <img src="../assets/images/wulf.png">
                </a>
            </div>
        </div>
    </div>
