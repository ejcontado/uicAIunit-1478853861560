JS & CSS dependency:

For tab buttons components, it will be ok to import following css file:
<link href="../wulfdist/css/wulf.basic.min.css" rel="stylesheet"/>
<link href="../wulfdist/css/wulf.buttons.min.css" rel="stylesheet"/>

You can also group buttons together to form button groups. This is done by wrapping the buttons to be grouped in a container with the class btn-group. You can also add n-tab-buttons to enable tab button behavior: 



    <div class="btn-group n-tab-buttons">
        <button class="btn btn-standard">Button</button>
        <button class="btn btn-standard">Button</button>
    </div>
     <div class="btn-group n-tab-buttons">
        <button class="btn btn-standard" disabled>Button</button>
        <button class="btn btn-standard" disabled>Button</button>
    </div>
    <div class="btn-group n-tab-buttons">
        <button class="btn btn-small">Button</button>
        <button class="btn btn-small">Button</button>
    </div>

Note that you can use disabled buttons and call to action versions of buttons with button groups as well.
