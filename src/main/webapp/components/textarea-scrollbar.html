Text area with scrollbar

Add class content-scroll in div, which has textarea child that has class n-textarea. Also you can change the width and height of textarea by overriding class name ".content-scroll" if you want as in following example.


    <form role="form">
        <div class="form-group">
            <div class="content content-scroll">
                <label for="ta-scroll">Scrolling text area</label>
                <textarea id="ta-scroll" class="n-textarea">TEXT TEXT TEXT TEXT TEXT TEXT TEXT</textarea>
            </div>
        </div>
    </form>

    // Change the textarea width and height if you want
    <style>
        .content-scroll{width:200px; height:200px;}
    </style>
