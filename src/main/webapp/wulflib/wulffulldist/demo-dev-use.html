Add WULF to your project


  1. Download the WULF distribution zip file from the Release Network Drive (see the page 1 Status and Plan for main content of each release)
  2. Unzip the distribution package
  3. Copy the wulfdist folder into your project's root folder
    • Note: the WULF full distribution package (wulffulldist) contains also demo related files in the root folder.
      Those files are not explicitly required in your web project, they are just for reference and demo purposes (feel free to utilize them if you want).
  4. Link the needed WULF files (css and js) into your HTML pages, for example: 
    <link href="wulfdist/css/wulf.min.css" rel="stylesheet">
...your html page content...
    Then:
    <script src="wulfdist/js/dependencies/jquery.min.js"></script>
<script src="wulfdist/js/dependencies/bootstrap.min.js"></script>
<script src="wulfdist/js/dependencies/jquery.mCustomScrollbar.concat.min.js"></script>
<script src="wulfdist/js/wulf.min.js"></script>
...your javascript files...
    OR
    <script src="wulfdist/js/dependencies/wulf-concat-deps.min.js"></script>
...your javascript files...
    As wulf-concat-deps.min.js contains bootstrap.min.js, jquery.mCustomScrollbar.concat.min.js and wulf.min.js, you can use that if you want.
  5. Add the needed extra dependencies (both css and js) for the complex components - see them from the component examples in the demo
  6. Optimize (clean up) unnecessary files away
    • Note: "wulfdist/css/images" package is used by jqWidgets. You can delete it if your project do not use those widgets at all.
See the 'Examples' tab in this demo if you are not clear how to use the components.
