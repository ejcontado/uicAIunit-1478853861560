JS & CSS dependency:

For tooltip components, it will be ok to import following css file:
<link href="../wulfdist/css/wulf.basic.min.css" rel="stylesheet"/>
<link href="../wulfdist/css/wulf.tooltip.min.css" rel="stylesheet"/>
Data-placement can be: top | bottom | left | right + left empty + optionally also auto to allow automatic positioning on the opposite side near screen edges. 


	<!-- Basic tooltip -->
	Some <a class="link" data-toggle="tooltip" data-placement="bottom" href="#" title="explanation text">text</a> to explain more with tooltip.

	<!-- JS-script needed -->
	<script>
	   $(function () {
		  $('[data-toggle="tooltip"]').tooltip();
	   })
	</script>
Option: container. Appends the tooltip to a specific element. Example: container: 'body'. This option is particularly useful in that it allows you to position the tooltip in the flow of the document near the triggering element - which will prevent the tooltip from floating away from the triggering element during a window resize. 

	<!-- JS-script -->
	<script>
	  $(function () {
	    $('[data-toggle="tooltip"]').tooltip({container: 'body', delay:500});
	  })
	</script>
