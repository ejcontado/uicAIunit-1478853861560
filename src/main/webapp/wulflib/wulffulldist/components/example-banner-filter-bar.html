Nokia Banner(filter bar)

Nokia Banner(filter bar) is based on the Nokia Banner component and it has a filter bar to confirm some information, such as time and location.

          Click <li> infilter-bar-nav to open or close filter bar.
          The filter-bar-setting-item item changes when select all lists and then hover on it.
          Add filter list in filter-bar-content and add filter title in filter-bar-footer.

          Use loadCountries function in navfilterbar.js to load countries and cities: 

                  
    <script>
        require(['jquery', 'wulf/navbar', 'wulf/list-group', 'wulf/navfilterbar'], function ($) {
            $(function(){
                $.getJSON("../wulfdist/js/countries-cities.json", function (countriesCities) {
                    $("#nokia_banner").loadCountries(countriesCities);
                });
            });
        });
    </script>
