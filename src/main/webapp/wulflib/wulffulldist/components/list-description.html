JS & CSS dependency:

For description list (dl/dt/dd) components, it will be ok to import following css file:
<link href="../wulfdist/css/wulf.basic.min.css" rel="stylesheet"/>
<link href="../wulfdist/css/wulf.list.min.css" rel="stylesheet">
