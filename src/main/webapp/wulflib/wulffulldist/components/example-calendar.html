      If you provide data-markup attribute as the example, the HTML markups of calendar will be generated automatically next to the input element which provides the data-markup attribute. Otherwise you should specify your HTML markup by yourself. We have prepared HTML markups for you as following. Please be noted that if you use the data-markup to generate the HTML markups automatically, you should not use 'data-initialize' to initialize the calendar at the same time.

      If you put your HTML markups by yourself, you can do the same thing as the first demo to initialize datepicker by passing your options as following. 

              
    <div id="calendar_date" class="datepicker fuelux">
        <div class="input-group n-calendar">
            <input class="form-control n-inputfield" id="myDatepickerInput1" name="myDatepickerName" type="text"/>
            *Copy the HTML markups to here*
        </div>
    </div>

    <script>
    $(document).ready(function () {
        $('#calendar_datetime').datepicker({
        allowPastDates: true,
        momentConfig: {
        culture: 'en',
        format: 'DD.MM.YYYY HH:mm:ss'
        }
        });
    });
    </script>

      To disable the calender, you can simply add disabled attribute to the element and if you are using the markup-API, please also update the data-markup. 

              
    <div id="disabled_calendar_date" class="col-md-9 datepicker fuelux">
        <div class="input-group n-calendar">
            <input class="form-control n-inputfield" id="myDisabledDatepickerInput1" name="myDatepickerName"
                   data-markup="disabled_calendar" type="text" disabled>
        </div>
    </div>

    <div class="input-group n-calendar">
        <input class="form-control n-inputfield" id="myDisabledDatepickerInput2" name="myDatepickerName"
               data-markup="calendar" type="text" disabled>

      To disable the past days in calendar, you can add class 'n-calendar-lock-past' to the calendar element. 

              
    <div class="input-group n-calendar n-calendar-lock-past">
        <input class="form-control n-inputfield" id="myDatepickerInputLocked" name="myDatepickerName"
               data-markup="calendar" type="text">
    </div>

      Using calendar with date range, you need two calendar components using class 'n-data-range-start'and class 'n-data-range-end', respectively. 

              
    <div id="calendar_date_range" class="n-data-range">
        <div class="datepicker fuelux">
            <div class="input-group n-calendar n-data-range-start">
                <input class="form-control n-inputfield" id="dateRangeInput1" name="dataRangeName1"
                       data-markup="calendar" type="text">
            </div>
        </div>
        <div class="datepicker fuelux">
            <div class="input-group n-calendar n-data-range-end">
                <input class="form-control n-inputfield" id="dateRangeInput2" name="dataRangeName2"
                       data-markup="calendar" type="text">
            </div>
        </div>
    </div>
