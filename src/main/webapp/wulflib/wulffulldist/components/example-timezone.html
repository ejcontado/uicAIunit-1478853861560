Timezone

Normal

JS & CSS dependency:

For timezone components, it will be ok to import following css file:
<link href="../wulfdist/css/wulf.basic.min.css" rel="stylesheet"/>
<link href="../wulfdist/css/wulf.buttons.min.css" rel="stylesheet"/>
<link href="../wulfdist/css/wulf.inputfield.min.css" rel="stylesheet"/>
<link href="../wulfdist/css/wulf.dropdowns.min.css" rel="stylesheet"/>
<link href="../wulfdist/css/wulf.timezone.min.css" rel="stylesheet">

Invoke Timezone.js by Required JS:


    <script src="../assets/js/require.js"></script>
    <script src="../assets/js/require-config.js"></script>
    <script>
    require(['jquery', 'wulf/timezone']);
    </script>

Via Javascript

Call the timezone control via JavaScript:


    $('mySelectlistTimezone').nTimezone();

Via Data-API

Add data-markup="timezone" to the .n-timezone element that you wish to initialize on $.ready().


    <div class="btn-group selectlist selectlist-resize n-timezone" data-resize="none" data-markup="timezone" id="mySelectlistTimezone">
        <button class="btn btn-default dropdown-toggle" data-toggle="dropdown" type="button" aria-expanded="false">
            <span class="selected-label"></span>
            <span class="selected-caret"><span class="caret"></span></span>
            <span class="sr-only">Toggle Dropdown</span>
        </button>
        <ul class="dropdown-menu n-dropdown-menu-scroll" role="menu">
        </ul>
    </div>

Methods

.setDefault(PackedZoneString)

Set the default timezone to show.


    $('mySelectlisttimezone').setDefault('Asia/Shanghai');

.addZone(PackedZoneString)

.addZone(PackedZoneString[])

Add more timezone to list. The strings passed in should be in this format: ZoneName|ZoneOffset.


    $('mySelectlisttimezone').addZone('Asia/Beijing|+08:00');
    $('mySelectlisttimezone').addZone(['Asia/Beijing|+08:00', 'Europe/Oulu|+03:00']);

.removeZone(PackedZoneString)

.removeZone(PackedZoneString[])

Remove the timezone in list.


    $('mySelectlisttimezone').removeZone('Asia/Beijing');
    $('mySelectlisttimezone').removeZone(['Asia/Beijing', 'Asia/Shanghai']);
