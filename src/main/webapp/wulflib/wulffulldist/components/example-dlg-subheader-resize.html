JS & CSS dependency:

For resize and drag dialog components, it will be ok to import following css file:
<link href="../wulfdist/css/wulf.basic.min.css" rel="stylesheet"/>
<link href="../wulfdist/css/wulf.buttons.min.css" rel="stylesheet"/>
<link href="../wulfdist/css/wulf.dialog.min.css" rel="stylesheet"/>

To enable the dialogs dragable, see the examples below. 


    <div class="modal-header" data-drag="True">

To enable the dialogs resizable, see the examples below. 


    <div class="modal-content" data-resize="True">
    <!----need change dialog "modal-header" structure to support resize-->
