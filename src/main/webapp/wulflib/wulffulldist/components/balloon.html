JS & CSS dependency:

For balloon components, it will be ok to import following css file:
<link href="../wulfdist/css/wulf.basic.min.css" rel="stylesheet"/>
<link href="../wulfdist/wulf.tooltip.min.css" rel="stylesheet"/>
<link href="../wulfdist/wulf.balloon.min.css" rel="stylesheet"/>

Invoke Balloon.js by Required JS

    
    <script src="../assets/js/require.js"></script>
    <script src="../assets/js/require-config.js"></script>
    <script>
        require(['jquery', 'wulf/balloon']);
    </script>

Via Javascript

Call the balloon control via JavaScript:

    
    $('[data-toggle="balloon"]').nBalloon();

Tips

1. Data-placement can be: top | bottom | left | right + also empty + optionally also auto to allow automatic positioning on the opposite side near screen edges.

2. Please note that Nokia GUI rule is to limit the size of tooltip to ONLY 2 lines.
