Column A Column B Column C

Item 1 Item 2 Item 3

JS & CSS dependency:

For multi-column lists components, it will be ok to import following css file:
<link href="../wulfdist/css/wulf.basic.min.css" rel="stylesheet"/>
<link href="../wulfdist/css/wulf.list.min.css" rel="stylesheet">
To make use of the multicolumn list, create a table with the class n-multicolumn-list. 

    <table class="n-multicolumn-list">
    <thead>
    <tr>
    <th>Column A<span class="icon icon-arrow"></span></th>
    <th>Column B<span class="icon icon-arrow"></span></th>
    <th>Column C<span class="icon icon-arrow"></span></th>
    </tr>
    </thead>
    <tbody>
    <tr>
    <td><span>Item 1</span></td>
    <td><span>Item 2</span></td>
    <td><span>Item 3</span></td>
    </tr>
    ...
    </tbody>
    </table>

Multicolumn list with subheaders
Column A Column B Column C
Subheader 1
Item 1 Item 2 Item 3
Subheader 2
Item 1 Item 2 Item 3
Item 1 Item 2 Item 3
Subheader 3
Item 1 Item 2 Item 3
To put subheaders, first, make use of the classes in the standard multicolumn list. The tr element must have the class subheader, its first td element occupying the whole row by using the colspan attribute. The items under each subheader must be contained in another tr element with the class subheader-item. 

    <table class="n-multicolumn-list">
    <thead>
    <tr>
    <th>Column A<span class="icon icon-arrow"></span></th>
    <th>Column B<span class="icon icon-arrow"></span></th>
    <th>Column C<span class="icon icon-arrow"></span></th>
    </tr>
    </thead>
    <tbody>
    <tr class="subheader">
    <td colspan="3" ><span class="icon icon-next"></span><span>Subheader
    1</span></td>
    </tr>
    <tr class="subheader-item">
    <td><span>Item 1</span></td>
    <td><span>Item 2</span></td>
    <td><span>Item 3</span></td>
    </tr>
    ...
    </tbody>
    </table>

Multicolumn rows
Column A Column B Column C
Item 1 Item 2 Item 3
Group Group Item 1 Group Item 1
Group Item 2 Group Item 2
Group Item 3 Group Item 3
Following the template for standard multicolumn lists, create a separate tbody element with the class group. The first td of the group must be of class n-multicolumn-list-td-group. This first td must occupy the whole column in the group by using the rowspan attribute. 

    <table class="n-multicolumn-list">
    <thead>
    <tr>
    <th>Column A<span class="icon icon-arrow"></span></th>
    <th>Column B<span class="icon icon-arrow"></span></th>
    <th>Column C<span class="icon icon-arrow"></span></th>
    </tr>
    </thead>
    <tbody>
    <tr>
    <td><span>Item 1</span></td>
    <td><span>Item 2</span></td>
    <td><span>Item 3</span></td>
    </tr>
    ...
    </tbody>
    <tbody class="group">
    <tr>
    <td rowspan="3"><span>Group</span></td>
    <td><span>Group Item 1</span></td>
    <td><span>Group Item 1</span></td>
    </tr>
    ...
    </tbody>
    </table>
