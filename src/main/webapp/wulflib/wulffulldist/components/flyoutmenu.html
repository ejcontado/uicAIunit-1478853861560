Flyout menu

  • Customer Care Automation
  • Troubleticket System
  • Map Tool
  • Intranet
  • Knowledge Database

When available, the flyout menu is indicated by a vertical tab positioned along the left edge of the interface. It expands to the right.

JS & CSS dependency:

For flyoutmenu components, it will be ok to import following css file:
<link href="../wulfdist/css/wulf.basic.min.css" rel="stylesheet"/>
<link href="../wulfdist/css/wulf.flyout.min.css" rel="stylesheet">

    <div class="n-flyout" id="myFlyout">
    <div class="n-flyout-container">
        <ul class="n-flyout-menu">
            <li><a href="javascript:void(0)"><span>Customer Care Automation</span></a></li>
            <li><a href="javascript:void(0)"><span>Troubleticket System</span></a></li>
            <li><a href="javascript:void(0)"><span>Map Tool</span></a></li>
            <li><a href="javascript:void(0)"><span>Intranet</span></a></li>
            <li><a href="javascript:void(0)"><span>Knowledge Database</span></a></li>
        </ul>
    </div>
    <a class="n-flyout-open" href="javascript:void(0)">
        <span class="middle"><i class="icon icon-list"></i></span>
    </a>
    </div>

To pre-select a list item, add the class "selected". Otherwise, the top-most item will be selected. 


    <li class="selected"><a href="javascript:void(0)"><span>Map Tool</span></a></li>

For AJAX generated scripts, the tooltip can be initialized via Javascript: 


    $("#myNewFlyout").nFlyout();
