For slider components, it will be ok to import following css file:
<link href="../wulfdist/css/wulf.basic.min.css" rel="stylesheet"/>
<link href="../wulfdist/css/wulf.slider.min.css" rel="stylesheet"/>

A basic slider is provided for selecting a number from a range of valid numbers. To use the slider, insert an input element with the type range: 


    <input type="range" min="0" max="10">

Note that you must include JavaScript to make the slider look and behave correctly, as the browser support for the native input varies. To include the JavaScript dependencies needed for the slider to work, just include wulf.min.js with your page. The script will automatically handle constructing and styling the slider on page load.

Listening for changes of the slider value can be done the usual way: 


    $('#slider').on('change input', function() {

    //Do things here with the value of the input slider
    var value = $(this).val();
    });
