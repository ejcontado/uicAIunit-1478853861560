Alert

Information alerts can be used to display errors or other messages with prominence.

Something has gone terribly wrong.
We might have a problem somewhere.
Operation finished successfully. This long text is for testing purposes to demonstrate a really really really long text that wraps at some point to another line in the message area.
No tick icon at the end of the alert, not closing this alert. Here is some long text to demonstrate behavior in text wrap situation.
This can be closed by the tick at the end of this alert.
Option to make alert without the close option.

JS & CSS dependency:

For alert components, it will be ok to import following css file:
<link href="../wulfdist/css/wulf.basic.min.css" rel="stylesheet"/>
<link href="../wulfdist/css/wulf.alert.min.css" rel="stylesheet"/>

To use alerts, wrap the alert message in a container marked with the class alert. You can further specialize the alert by using alert alert-error, which will result in an error alert with red text and a bold font. You can add a custom Nokia icon to symbolize the type of the alert, i.e. a warning icon or an information icon by using icons like <span class="icon icon-info"></span> (options are icon-ok, icon-warning, icon-info, icon-error). 


    <!-- Standard alert -->
    <div class="alert">
        <span class="icon icon-info"></span>We might have a problem somewhere
        <a href class="n-close" data-dismiss="alert" role="button" aria-label="Alert close button">
            <span class="icon icon-close" aria-hidden="true"></span>
        </a>
    </div>

    <!-- Error alert -->
    <div class="alert alert-error">
        <span class="icon icon-error"></span>Something has gone terribly wrong.
        <a href class="n-close" data-dismiss="alert" role="button" aria-label="Alert close button">
            <span class="icon icon-close" aria-hidden="true"></span>
        </a>
    </div>

    <!-- Dismissable alert -->
    <div class="alert alert-error alert-dismissible">
        <span class="icon icon-error"></span>This can be closed by the tick at the end of this alert.
        <a href class="n-close" data-dismiss="alert" role="button" aria-label="Alert close button">
            <span class="icon icon-close" aria-hidden="true"></span>
        </a>
    </div>
