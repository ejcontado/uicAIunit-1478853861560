Tips

In order to show scrollbar correct, the scrollbar will be auto updated per 60ms as default.
You can switch on/off the auto updating by yourself.

Two ways to switch off auto updating
   1. Add classname "scrollbar-not-autoupdate" together with "n-*-scroll"
   2. Call nScrollbar with option: {notAutoUpdate:true}

Example:
   1. Using classname:
       <table class="n-table n-table-standard n-table-striped n-table-cell-hover">
            <thead>...</thead>
            <tbody class="n-table-scrollbar scrollbar-not-autoupdate">...</tbody>
       </table>

   2. Using nScrollbar()
       $(document).ready(function(){
           $(".n-table-scrollbar").nScrollbar({notAutoUpdate:true});
           $(".n-dropdown-menu-scroll").nScrollbar({notAutoUpdate:true});
           ......
       })

You need to update scrollbar once data is added/removed if auto Updating is switched off.
   Example:
     $(".n-table-scrollbar").nScrollbar("update");
     $(".n-dropdown-menu-scroll").nScrollbar("update");
     ....
