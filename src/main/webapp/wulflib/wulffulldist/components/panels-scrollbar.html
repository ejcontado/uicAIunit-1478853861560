JS & CSS dependency: 
For Panel with scrollbar components, it will be ok to import following css file:
<link href="../wulfdist/css/wulf.basic.min.css" rel="stylesheet"/>
<link href="../wulfdist/css/wulf.panel.min.css" rel="stylesheet">

You can get basic panel scroll by setting into a div container a class panel-body-scroll: 


  <div class="panel panel-shadow panel-blue-cap">
  <div class="panel-heading">
    <h1>Available Operations</h1>
  </div>
  <div class="panel-body panel-body-scroll">
    ...
  </div>
  </div>
  <script>
    $(document).ready(function(){
      $(".panel-body-scroll").nScrollbar();
    });
  </script>
