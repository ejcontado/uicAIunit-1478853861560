Button

Regular buttons

JS & CSS dependency:

For button components, it will be ok to import following css file:
<link href="../wulfdist/css/wulf.basic.min.css" rel="stylesheet"/>
<link href="../wulfdist/css/wulf.buttons.min.css" rel="stylesheet">

You can use a number of different Nokia-style buttons by adding a btn class definition to a button element. A number of different button sizes are provided through additional size classes like btn-large, btn-standard and btn-small. 


    <button class="btn btn-large">Button</button>
    <button class="btn btn-standard">Button</button>
    <button class="btn btn-small">Button</button>
    <button class="btn btn-standard" disabled>Button</button>

Buttons dark

You can also use a dark version of the buttons by adding the btn-action class to the button element. These buttons are stylistically distinctive to encourage the user to take action. 


    <button class="btn btn-action btn-standard">Button</button>
    <button class="btn btn-action btn-standard" disabled>Button</button>

Not print buttons

You can add .btn-not-print for button if you don't want it to be shown when printing.

<button class="btn btn-large btn-not-print">Button</button>

Special buttons

In addition to the regular buttons, you can also choose from a few different special buttons for common use cases, such as closing, minimizing,accessing help and displaying information. These buttons do not require any additional markup other than adding a class like btn-close, btn-minimize, btn-help or btn-info : 


    <button class="btn btn-close"></button>
    <button class="btn btn-minimize"></button>
    <button class="btn btn-help"></button>
    <button class="btn btn-info"></button>

Note that these buttons are defined by appereance only and they do not provide any functionality by themselves. You should use JavaScript if you wish to define functionality for the buttons.

Button without text

You can use a icons inside buttons as well by including a span element with the icon definition. To make sure that the button dimensions are correct, add the btn-icon class to the button: 


    <button class="btn btn-icon">
        <span class="icon icon-settings"></span>
    </button>
    <button class="btn btn-icon" disabled>
        <span class="icon icon-refresh"></span>
    </button>

If you wish to include also text alongside an icon in the button, make sure to wrap your text content in a span to ensure proper padding between the elements: 


    <button class="btn btn-icon">
        <span class="icon icon-bubble"></span>
        <span>Buttons with icons</span>
    </button>

Default Button

You can turn a button to default button. To make sure that the button dimensions are correct, add the btn-defaultBlue class to the button: 


    <button class="btn btn-standard btn-defaultBlue">
    </button>
    <button class="btn btn-close btn-defaultBlue">
        <span class="icon icon-close"></span>
    </button>
    <button class="btn btn-icon btn-defaultBlue">
        <span class="icon icon-bubble"></span>
    </button>

Carousel buttons

Chrome style Carousel button usage:

        <span class="icon icon-back"></span>
    </button>
    <button class="btn btn-carousel-chrome-next">
        <span class="icon icon-next"></span>
    </button>
    <button class="btn btn-carousel-chrome-back" disabled>
        <span class="icon icon-back"></span>
    </button>
    <button class="btn btn-carousel-chrome-next" disabled>
        <span class="icon icon-next"></span>
    </button>;
