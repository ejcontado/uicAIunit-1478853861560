Gray text links can function as a lower level navigation within the interface. Use nav-local as base for all localized sub navigation. To use normal menu on a panel, use nav-local-menu as a class into the ul. And add nav-local-menu-item for each li. Use nav-local-menu-divided for the version with divider.

            
    <!-- Normal localized sub navigation menu-->
    <div class="panel panel-shadow">
        <!-- Panel Heading-->
        <ul class="nav-local nav-local-menu">
            <li class="nav-local-menu-item">
                <a href="#">
                    <span class="icon icon-home"></span>
                    <span>Message Board Home></span>
        <!-- Closing tags -->
    </div>

    <!-- Divided localized sub navigation menu-->
    <ul class="nav-local nav-local-menu-divided">
        <li class="nav-local-menu-item">
            <a href="#">
                <span class="icon icon-home"></span>
                <span>Message Board Home></span>
            </a>
        </li>
    </ul>

    Requires: localized-navigation.js or wulf.js

The interactive panel always expands below the selected element. Theinline toolbaris the Additional navigation. The Inline toolbar is a simplified variation on the drill-down pattern. So we use n-drillDown-item. It opens from the bottom of the selected element. So we set the direction as data-target-selector="#item1". We can integrate it with drilldown component and The component should point to the content with id:id="item1". 


<!-- The Inline toolbar of the localized sub navigation menu.-->
<div class="nav-local-inline-toolbar">

     <div class="trigger">
       <h1>Balance</h1>
         <div class="description">Maecenas sed diam ege</div>
         <button class="btn btn-default n-drillDown-item"  data-target-selector="#item1"></button>
     </div>
     <div class="items-icon">

</div>

Additional navigation may be necessary to access details within panels on a screen. Use the horizontal navigation first, then add the vertical navigation if another layer of information is required. Use nav-local-horizontal for horizontal navigation and nav-local-vertical for vertical navigation. 


    <!-- Horizontal localized sub navigation-->
    <!-- Add nav-local-horizontal-row for the row style-->
    <div class="row nav-local-horizontal-row">
        <ul class="nav-local nav-local-horizontal">
            <li>
                <a href="#">
                    <span>Item One></span>
                </a>
            </li>
        </ul>
    </div>

    <!-- Vertical localized sub navigation-->
    <ul class="nav-local nav-local-vertical">
        <li>
            <a href="#">
                <span>Item One></span>
            </a>
        </li>
    </ul>

    To make the left corner radius rounded, add nav-local-vertical-round-left on the ul.
    <!-- Rounded Vertical local sub navigation -->
    <ul class="nav nav-local-vertical nav-local-vertical-round-left">

    Requires: localized-navigation.js or wulf.js
