JS & CSS dependency: 
For Tabbed pane components, it will be ok to import following css file:
<link href="../wulfdist/css/wulf.basic.min.css" rel="stylesheet"/>
<link href="../wulfdist/css/wulf.tabpane.min.css" rel="stylesheet">
<link href="../wulfdist/css/wulf.links.min.css" rel="stylesheet">
If you want to have other than the first tab initially shown, you define: class="active" for the li and div.tab-pane and also a js script to pick it at document loading. 

    <!-- Basic tabs (navs) -->
    <ul class="nav nav-tabs">
    <li role="presentation" class="active"><a href="#A" data-toggle="tab">Data A</a></li>
    <li role="presentation"><a href="#B" data-toggle="tab">Data B</a></li>
    <li role="presentation"><a href="#C" data-toggle="tab">Data C</a></li>
    <li role="presentation" class="disabled"><a data-toggle="tab" disabled="disabled">Data D</a></li>
    </ul>
    <div class="tab-content">
    <div class="tab-pane active" id="A">Content inside tab A</div>
    <div class="tab-pane" id="B">Content inside tab B</div>
    <div class="tab-pane" id="C">Content inside tab C</div>
    <div class="tab-pane active" id="D">Content inside tab D</div>
    </div>

To add close icon on each tab, add <a href="javascript:void(0)" class="n-close"><span class="icon icon-close"></span></a>
         
    <!-- Basic tabs (navs) -->
    <ul class="nav nav-tabs">
    <li role="presentation" class="active"><a href="#dataA" data-toggle="tab">Data A</a><a href="javascript:void(0)" class="n-close"><span class="icon icon-close"></span></a></li>
    <li role="presentation"><a href="#dataB" data-toggle="tab">Data B</a><a href="javascript:void(0)" class="n-close"><span class="icon icon-close"></span></a></li>
    <li role="presentation"><a href="#dataC" data-toggle="tab">Data C</a><a href="javascript:void(0)" class="n-close"><span class="icon icon-close"></span></a></li>
    <li role="presentation" class="disabled"><a data-toggle="tab" disabled="disabled">Data D</a><a href class="n-close"><span class="icon icon-close"></span></a></li>
    </ul>
    <div class="tab-content">
    <div class="tab-pane active" id="dataA">Content inside tab A</div>
    <div class="tab-pane" id="dataA">Content inside tab B</div>
    <div class="tab-pane" id="dataA">Content inside tab C</div>

    </div>
