JS & CSS dependency:

For Scroll components, it will be ok to import following css file:
<link href="../wulfdist/css/wulf.basic.min.css" rel="stylesheet"/>

Add the content needs to scroll under the <div> you use nScrollbar 


    <!-- Main scroll setting -->
    <div id="demo" class="n-scroll">
        <!-- Your content to be scrolled -->
    </div>

Vertical text scroll 


    <!-- Vertical text scroll -->
    <!-- JS-script needed -->
    <script>
        $(document).ready(function () {
            $("#demo").nScrollbar();
        });
    </script>

Horizontal text scroll
Set the option parameter axis to 'x' in javascript. Set the option parameter advanced: {autoExpandHorizontalScroll: true} in javascript. 


    <!-- Horizontal text scroll -->
    <!-- JS-script needed -->
    <script>
        $(document).ready(function () {
            $("#demo-x").nScrollbar(
            axis:"x",
            advanced:{autoExpandHorizontalScroll:true});
        });
    </script>

Both Vertical and Horizontal scroll at the same time
Set the option parameter axis to 'yx' in javascript. Set the option parameter advanced: {autoExpandHorizontalScroll: true} in javascript. 


    <!-- Vertical and Horizontal text scroll -->
    <!-- JS-script needed -->
    <script>
        $(document).ready(function () {
            $("#demo-yx").nScrollbar({
            axis:"yx",
            advanced:{autoExpandHorizontalScroll:true}});
        });
    </script>

Disabled scroll
Set the option parameter alwaysShowScrollbar to 2 in javascript. Set the option parameter theme to disabled in javascript. Set the option parameter mouseWheel to enable: false in javascript. Set option parameter disable in nScrollbar() method. 


    <!-- Text scroll(disabled) -->
    <!-- JS-script needed -->
    <script>
        $("#demo-yx-disabled").nScrollbar({
            axis:"yx",
            advanced:{autoExpandHorizontalScroll:true},
            alwaysShowScrollbar:2,
            theme:"disabled"
            mouseWheel:{ enable: false }
        });
        $("#demo-yx-disabled").nScrollbar("disable");
    </script>
