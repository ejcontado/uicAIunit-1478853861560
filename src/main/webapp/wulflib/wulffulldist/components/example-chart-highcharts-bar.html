Bar Charts (HighCharts)

Bar - horizontal:

Note: you need to have purchased the HighCharts license and placed the library files under wulfdist/js/dependencies folder to get the chart visible.

    <!-- Charts -->
    <div id="container" style="width:100%; height:400px;"></div>

    <!-- Needed scripts -->
    <script src="../wulfdist/js/dependencies/jquery.min.js"></script>
    <script src="../wulfdist/js/dependencies/highcharts.src.latest.js"></script>
    <script src="../wulfdist/js/dependencies/charts.js"></script>
