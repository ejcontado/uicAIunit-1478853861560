Text area

JS & CSS dependency:

For text area components, it will be ok to import following css file:
<link href="../wulfdist/css/wulf.basic.min.css" rel="stylesheet"/>
<link href="../wulfdist/css/wulf.textarea.min.css" rel="stylesheet">

Add class n-textarea under the textarea element:


    <!-- Text area -->
    <form role="form">
        <div class="form-group">
            <label for="ta-normal">Normal text area</label>
            <textarea id="ta-normal" class="n-textarea">
                TEXT TEXT TEXT TEXT TEXT TEXT TEXT
            </textarea>
        </div>
    </form>

Add class n-textarea under the textarea element and disabled label after the element:


    <!-- Text area (disabled) -->
    <form role="form">
        <div class="form-group">
            <label for="ta-disabled" class="control-label label-disabled">Disabled textarea</label>
            <textarea id="ta-disabled" class="n-textarea" disabled>
                TEXT TEXT TEXT TEXT TEXT TEXT TEXT
            </textarea>
        </div>
    </form>

Add class n-textarea under the textarea element and readonly label after the element:


    <!-- Text area (read-only) -->
    <form role="form">
        <div class="form-group">
            <label for="ta-readonly">Readonly textarea</label>
            <textarea id="ta-readonly" class="n-textarea" readonly>
                TEXT TEXT TEXT TEXT TEXT TEXT TEXT
             </textarea>
        </div>
    </form>

Add class content-scroll in div, which has textarea child that has class n-textarea. Also you can change the width and height of textarea by overriding class name ".content-scroll" if you want as in following example.


    <form role="form">
    <div class="form-group">
    <div class="content content-scroll">
    <label for="ta-scroll">Scrolling text area</label>
    <textarea id="ta-scroll" class="n-textarea">TEXT TEXT TEXT TEXT TEXT TEXT TEXT</textarea>
    </div>
    </div>
    </form>

    // Change the textarea width and height if you want
    <style>
    .content-scroll{width:200px; height:200px;}
    </style>
