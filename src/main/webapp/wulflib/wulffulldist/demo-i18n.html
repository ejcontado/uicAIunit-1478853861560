i18n (Internationalization)

i18n has be included in WULF. It could make your project meet the needs of different languages.


Usage

Your project could using i18n by including some third-party js as follows:
angular-translate.js
angular-translate-loader-partial.js
The way getting values has a little change. the key should be mapped in language file: 
{{'key'|translate}}

Supported components at present:

  • Angular: Table with filters
  • Angular: Table with paging
