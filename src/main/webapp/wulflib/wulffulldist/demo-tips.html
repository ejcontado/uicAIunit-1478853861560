Removing flash of unstyled text

  • Font loader: In order to avoid FOUT (flash of unstyled text), you can use webfont.js. Please add the following JavaScript: 
    • <script src="https://ajax.googleapis.com/ajax/libs/webfont/1.5.18/webfont.js"></script>

<script>
WebFont.load({
custom: {
families: ['Nokia Pure:n3,n4,n7'],
urls : ['wulfdist/css/wulf.min.css'] // define the file location in your case
}
});
</script>

Jquery-ui theme tips:

  • Where to get the theme file:
"Download" -> "Download Releases" -> "Logical place to search for the theme downloading"

1.The demo page to show jquery-ui control with wulf theme style
  juery-ui-wulf\jquery-ui\index.html

2.The links that is need to be imported
  <link href="../wulfdist/css/wulf.basic.min.css" rel="stylesheet"/>
  <link href="jquery-ui.theme.css" rel="stylesheet">

3.What is wulf.basic.min.css?
  It is the basic style definition in wulf, such as font family(Nokia Pure), font size, layout.
  In jquery-ui.theme,  nokia font is needed, so wulf.basic.min.css and font folder is also included.
