Localized Sub Navigation



Overview

Angular version of localized sub navigation component. It was implemented by using .component() method introduced since Angular1.5. This component can be used to generate two kinds of localized sub navigations.

wf-subnav settings

Attributes:Bindings Accepted values Notes
type:<
  • icon
  • arrows
 Different appearance of localized sub navigation.
navTitle: < It is localized sub navigation's title.
tabs: < It is array about localized sub navigation's tabs.

Keyboard accessibility

Besides the basic keyboard support, e.g. Tab, Shift + Tab and Enter. This component also supports the following keystrokes for the interaction.
Interaction Keystrokes Notes
Move focus to next/previous tab Tab/Shift+Tab - next/previous tab
  • Using Tab orShift + Tab can navigate out of the group of tabs
Select tab Enter - Select current focused tab
  • The associated tab content would be also shown
 
                    


   


                    
                
                    
angular.module('wf.ng.demo', ['wf.angular.localizedSubnavigation']).controller('NavigationController', NavigationController);
    function NavigationController() {
        var newTab = function (tabName,icon,tabContent) {
            return {
                tabName: tabName,
                icon: icon,
                tabContent: tabContent
            }
        };
        var self = this;
        self.levelOnewithIcon = {};
        self.levelOnewithIcon.type = "icon";
        self.levelOnewithIcon.navTitle = "Message Board";
        self.levelOnewithIcon.tabs = [  newTab('Message Board Home', 'icon-home','content 1'),
                                newTab('My Posts', 'icon-profile', 'content 2'),
                                newTab('New Posts', 'icon-add', 'content 3'),
                                newTab('setting', 'icon-settings','content 4')
                             ];
    }
