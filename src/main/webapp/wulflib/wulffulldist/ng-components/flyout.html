Flyout menu

Overview

Angular version of flyout component. It contains two components wfFlyout and wfFlyoutItem which are implemented by using .component() method introduced since Angular1.5.

CSS dependency:

For flyoutmenu components, it will be ok to import following css file:
<link href="../wulfdist/css/wulf.basic.min.css" rel="stylesheet"/>
<link href="../wulfdist/css/wulf.ngflyout.min.css" rel="stylesheet">

wf-flyout settings

Attributes:Bindings Accept value Notes
direction: @
  • right (Default)
  • left
 Defined the flyout direction,e.g.if the value is right, it will fly out from left to right.

wf-flyout-item settings

Attributes:Bindings Accept value Notes
value: @ String flyout item value
on-click-callback: & function (event) Callback function for click event. It will be called after you clicked the item of flyout.

Keyboard accessibility

Besides the basic keyboard support, e.g. Tab, Shift + Tab and Enter. This component supports the following keystrokes for the interaction.
Interaction Keystrokes Notes
Move focus to next/previous flyout item ↓ or ↑ - next/previous flyout item
Select flyout item Enter - Select current focused item
  • The disabled item cannot be selected

Missing functionality

  • Scrollbar not support yet.
