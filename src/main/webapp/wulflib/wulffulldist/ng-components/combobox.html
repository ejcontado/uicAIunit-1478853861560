Overview

Angular version of tabs component. It contains two components wfCombobox and wfComboboxItem which are implemented by using .component() method introduced since Angular1.5. This component can be used to generate tab panes from both static and dynamic model data (using ng-repeat directive for wf-combobox).

wf-combobox settings

Attributes:Bindings Accept value Note
pattern: <
  • textfield (Default)
  • clearable-textfield
  • menu
  • sm-menu
 The pattern of combo box to be created.
disable: <?
  • true
  • false (Default)
 Disable the combo box or not.

wf-combobox-item settings

Attributes:Bindings Accept value Note
item-value: < String (Default is null) The value for combo box item.
item-text: < String (Optional) The text for combo box item to shown. If not set, the text will present as the item-value.
selected: <?
  • true
  • false (Default)
 Set the default selected item.
disable: <?
  • true
  • false (Default)
 Disable the combo box item or not.
on-item-selected: & function(itemValue, itemText, event) Callback function when combo box item was selected.
  • itemValue - the value for current combo box item
  • itemText - The text for current combo box item to shown
  • event - event when item is selected

Keyboard accessibility

Besides the basic keyboard support, e.g. Tab, Shift + Tab and Enter. This component also supports the following keystrokes for the interaction.
Interaction Keystrokes Notes
Move focus to next/previous item ↑ or ↓ - previous/next item
  • The disabled item would be skipped
  • Using Tab or Shift + Tab can navigate out of the group of items
Select item Spacebar/Enter - Select current focused item
  • The disabled item cannot be selected
Trigger clear button Spacebar/Enter - Click on clear button

Missing functionality

  • Scrollbar not support yet.
  • Tooltip not support yet.
                    
    


        
        
            
            
            
            
        

        
         
            
         
    

                    
                
                    
    angular.module('wf.ng.demo', ['wf.angular.combobox']).controller('ComboboxDemoController', ComboboxDemoController);

    function ComboboxDemoController() {
        this.textfiledCombobox = {};
        this.clearableTextfiledCombobox = {};
        this.menuCombobox = {};
        this.smallMenuCombobox = {};

        this.textfiledCombobox.pattern = 'textfield';
        this.clearableTextfiledCombobox.pattern = 'clearable-textfield';
        this.menuCombobox.pattern = 'menu';
        this.smallMenuCombobox.pattern = 'sm-menu';

        this.textfiledCombobox.disabled = false;
        this.menuCombobox.disabled = false;

        var onItemSelected = function(itemValue, itemText) {
            alert('item-value: '+itemValue+ '\n' +'item-text: ' + itemText );
        };
        this.items = [
            {value:'1', text:'One', disabled:false, selected:true, onItemSelected: onItemSelected},
            {value:'2', text:'Two', disabled:false, onItemSelected: onItemSelected},
            {value:'3', text:'Three', disabled:false, onItemSelected: onItemSelected},
            {value:'4', text:'Really Long Link Thing That Goes Wider Here', disabled:false, onItemSelected: onItemSelected}
        ];

    }
