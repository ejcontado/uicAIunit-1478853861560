Overview

Angular version of splitter component. It is implemented by using .component() method introduced since Angular1.5. This component consisting of a moveable split bar(s) that divides a container's display area into two or more resizable panels.

wf-splitter settings

Attributes:Bindings Accept value Note
splitter-style: < Object (Default: {width: '100%',height: '100%',background: '#ffffff'}) The style of the splitter container.
orientation: @
  • vertical(Default)
  • horizontal
 The split orientation of the splitter.
on-drag: & Function(pane, nextPane, event) Callback function when splitter bar was moved.
  • pane - html element object of first pane
  • nextPane - html element object of next pane
  • event - mouseevent when handler bar was moved

wf-split settings

Attributes:Bindings Accept value Note
init-size: @ Number/Number with 'px' unit/Percentage
(eg. 50/50px/50%) 		The initial size of the spit pane. For vertical orientation is width, for horizontal orientation is height. The default unit is 'px' when pass number value.
min-size: @ Number/Number with 'px' unit/Percentage
(eg. 50/50px/50%) 		The minimum size of the spit pane. For vertical orientation is width, for horizontal orientation is height. The default unit is 'px' when pass number value.

    angular.module('wf.ng.demo', ['wf.angular.splitter']).controller('splitterDemoController', splitterDemoController);
    function splitterDemoController() {
        var self = this;
        self.style = {
            width: '300px',
            height: '200px',
            background: '#ffffff',
            borderRadius: '6px',
            border: '1px solid #d8d8d8'
        };
        self.onVerticalDrag = function (pane1, pane2) {
            alert('pane offset will be changed to ' + pane1.style.width + '\nsibling pane offset will be changed to ' + pane2.style.width);
        };
        self.onHorizontalDrag = function (pane1, pane2) {
            alert('pane offset will be changed to ' + pane1.style.height + '\nsibling pane offset will be changed to ' + pane2.style.height);
        };
    }
    angular.element(document).ready(function () {
        angular.bootstrap(document, ['wf.ng.demo'], {
            strictDi: true
        });
    });
