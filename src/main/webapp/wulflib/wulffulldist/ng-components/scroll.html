Overview

Angular wrapper of Malihu's jQuery Custom Scrollbar which is implemented by using .directive() method introduced since Angular1.5

wf-scrollbar settings

Attributes:Bindings Accept value Note
scrollbar-config: < Object
  • height - set the height of content
  • width - set the width of content
  • axis - add 2-axis scrollbars with the set to 'x','y' or 'yx'.(Default: 'y')
  • disable - disabled scrollbar. (Default: false)
  • onScroll - callback function when content is scrolled.
 e.g.
{
 height: 300,
 width: 400,
 axis: 'yx',
 disable: false,
 onScroll: function(){
  console.log(angular.element(this).mcs) // mcs is a object which contains scroll element offset value
 }
}

Keyboard accessibility

This component also supports the following keystrokes for the interaction.
Interaction Keystrokes Notes
scroll up/down Up/ Down or Page up/ Page down
scroll left/right Left/ Right or Page up/ Page down
scroll top/bottom Home/ End 
                    


    

        
    



                    
                
                    
angular.module('wf.ng.demo', ['wf.angular.scrollbar']).controller('scrollDemoController', scrollDemoController);
    function scrollDemoController() {
        var self = this;
        self.config = {
            height: 300,
            width: 400,
            axis: 'yx',
            disable: false
            onScroll: function(){
                var scrollObj = angular.element(this);
                alert('This is an onScroll callback. \n scroll direction: ' + scrollObj[0].mcs.direction +'\n' +
                      'offset left: ' + scrollObj[0].mcs.left +'\n' +
                      'offset top: ' + scrollObj[0].mcs.top )
            }
        }
    }
angular.element(document).ready(function () {
    angular.bootstrap(document, ['wf.ng.demo'], {
        strictDi: true
    });
});
