Toggle Button

Standard
Disabled

Overview

Angular version of toggle button component which is implemented by using .component() method introduced since Angular1.5.

wf-toggle-button settings

Attributes:Bindings Accept value Note
toggle-id: @
  • valid html id string
 [required]Used for switch button, the id for toggle switch button
color: <
  • true
  • false(Default)
 [Optional]Used for switch button, if the toggle switch button will render with color
checked: <
  • true
  • false(Default)
 [Optional]Used for switch button, the enable state of toggle switch button
icon-display-style: @
  • 'btn-icon-button'
  • 'btn-icon'
 [required]Used for icon button, if toggle icon button display as icon or button
icon-class-name: @
  • icon class name(icon class name list)
 [Optional]Used for icon button, the class name of icon display in toggle icon button
text: @
  • valid string text
 [Optional]Used for icon button, the text display in toggle icon button
toggle-icon: <
  • true
  • false(Default)
 [Optional]Used for icon button, if the toggle is toggle icon button
emphasis: <
  • true
  • false(Default)
 [Optional]Used for icon button, if toggle icon button render with color when selected
disable: <
  • true
  • false(Default)
 [Optional]Used for all toggle button, the status of toggle button
size: @
  • 'small'
  • 'standard'(default value)
 [Optional]Used for all toggle button, the toggle button size
on-click: & function(event) Callback function when button is clicked
  • event - event when button is clicked

If you are using this component in CSP mode, please make sure the 'unsafe-inline' is enabled.
