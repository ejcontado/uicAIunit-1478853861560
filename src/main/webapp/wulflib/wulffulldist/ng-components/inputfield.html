Overview

We did not capsulate input text tag but just show how to use directive to update component status.
Detail angular input text information please refer to https://docs.angularjs.org/api/ng/directive/input.

AngularJS provides basic form validation by honoring the standard HTML element attributes, such as type and required, and adding some directives.
The directive that AngularJS uses for input elements provides some additional attributes that can be used to improve integration with the data model, as below:

  • ng-model— Specifies a two-model binding.
  • ng-change— Specifies an expression that is evaluated when the contents of the element are changed.
  • ng-minlength— Sets a minimum number of characters required for the element to be valid.
  • ng-maxlength— Sets a maximum number of characters required for the element to be valid.
  • ng-pattern— Sets a regular expression. The contents of the element must match this pattern in order to be valid.
  • ng-required— Sets the value of the required attribute with a data binding..
The form object comes with a few predefined states that allow you to make decisions on what actions you want to allow. These states are as follows:
  • $pristine— This Boolean flag indicates that the form is unmodified.
  • $dirty— This Boolean flag indicates that the form has been modified.
  • $valid— This Boolean flag indicates that the form is in a valid state.
  • $invalid— This Boolean flag indicates that the form is in an invalid state.
  • $error— This object contains all the validations on a form and whether they’re valid or invalid.
  • $touched— This Boolean flag indicates that a control has lost focus.
AngularJS also adds a corresponding CSS class to the form element, reports on the outcome of these validation checks by adding and removing the elements it validates from a set of classes, which can be combined with CSS to provide feedback to the user by styling the element:
  • .ng-valid— the model is valid.
  • .ng-invalid— the model is invalid.
  • .ng-valid-[key]— for each valid key added by $setValidity.
  • .ng-invalid-[key]— for each invalid key added by $setValidity.
  • .ng-pristine— the control hasn't been interacted with yet.
  • .ng-dirty— the control has been interacted with.
  • .ng-touched— the control has been blurred.
  • .ng-untouched— the control hasn't been blurred.
  • .ng-pending— any $asyncValidators are unfulfilled.
  • .ng-empty— the view does not contain a value or the value is deemed "empty", as defined by the ngModel.NgModelController method.
  • .ng-not-empty— the view contains a non-empty value.
You can define ng-pattern with yourself, use this directives and attributes and "ng-show", "ng-if" to complete validation.

angular.module('wf.ng.demo', ['wf.angular.tabs']).controller('InputfieldDemoController', InputfieldDemoController);
function InputfieldDemoController() {
    var newInstance = function (disable, readonly, content) {
        return {
            disable: disable,
            readonly: readonly,
            content: content
        }
    };

    var self = this;
    self.standard = newInstance(false, true, "Text");
    self.small = newInstance(false, true, "Text");
    self.validation = newInstance(false, true, "Text");

    self.standard.enableDisableShow = "Editable field";
    self.standard.updateEnableDisable = function () {
        self.standard.disable = !self.standard.disable;
    };

    self.standard.readonlyOperationShow = "Read-only input field";
    self.standard.updateReadonly = function () {
        self.standard.readonly = !self.standard.readonly;
    };

    self.small.enableDisableShow = "Editable field";
    self.small.updateEnableDisable = function () {
        self.small.disable = !self.small.disable;
    };

    self.small.readonlyOperationShow = "Read-only input field";
    self.small.updateReadonly = function () {
        self.small.readonly = !self.small.readonly;
    };

    self.validation.updateEnableDisable = function () {
        self.validation.disable = !self.validation.disable;
    };

    self.validation.initValue = "Changed";
    self.validation.result = "";
    self.validation.entryChange = "Changed";
    self.validation.noChange = function () {
        self.validation.result = angular.equals(self.validation.initValue,  self.validation.entryChange );
        return self.validation.result;
    }

    self.requireValue =  true;
    self.letterPattern = new RegExp("^[a-zA-Z]");
    self.digitalPattern = new RegExp("^[0-9]");
    self.minMaxPattern = new RegExp("^(0?[1-9]|[1-9][0-9])$");
    self.pattern2 = new RegExp("^[a-zA-Z0-9]");
}
