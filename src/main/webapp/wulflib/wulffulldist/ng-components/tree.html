Tree

Overview

Angular version of tree component. It contains two components wfTree and wfTreeNode which are implemented by using .component() method introduced since Angular1.5. This component can be used to generate tree from both static and dynamic model data (using ng-repeat directive for wf-tree-node).

wf-tree settings

Attributes:Bindings Default/Arguments Notes
treeData:= the content of tree The data for tree to load.like:[ { "title": 'Content', "children":[{}] }, { "title": 'Content', "children":[{}] }]
hasCheckbox: <? Default: false if true,there will have be a checkbox in every tree node for check;if false,tree node without checkbox.
disable: <? Default: false if true,current tree can do operate;if false,tree can not operate.

wf-tree-node settings

Attributes:Bindings Default / Arguments Notes
label: < String Label/title of tree node
isSelected: < Default: false Whether tree node is selected when tree node with checkbox.
children: < the children of current tree node. if current tree node has children,it can be expand and with expand icon.
hasCheckbox: < Default:false Whether current node is have checkbox.
node: < the current tree node. this is data of current node.

Keyboard accessibility

Besides the basic keyboard support, e.g. Tab, Shift + Tab and Enter. This component also supports the following keystrokes for the interaction.
Interaction Keystrokes Notes
Move focus to next/previous tree node →/↓ or ←/↑ - next/previous tree node
  • Using → or ← can expand/collapse the tree node if it has children.
  • Using ↓ or ↑ can choice the next/previous tree node.
  • Using Tab or Shift + Tab can navigate the next tree node
Select tree node Enter - Select current focused tree node
  • The associated tree node content would be also shown
