Login

Overview

Angular version of login component which is implemented by using .component() method introduced since Angular1.5.

wf-login settings

Attributes:Bindings Accept value Note
loginType: <
  • standard(default) - Login standard
  • signup - Login account sign-up
  • product - Login product
  • legalcopy - Login with integrated legal copy
  • acceptance - Login with legal acceptance
  • error - Login error
  • withfourcolumn - standard progress loading icon
 The login dialog type
title-text: < String: title text
version: < String: version text
copyright: < String: copyright text
product-text: < String: the product text which below the version It is just for login product dialog
legalcopy-text: < String: the legal copy text which below the login button It is just for legal copy dialog
acceptance-text: < String: the acceptance text which in acceptance dialog It is just for acceptance dialog
error-text: < String: the error text which below the version It is just for error dialog
on-login: & function(username, password, event) Callback function when log-in buttton is clicked.
  • username - value of username
  • password - value of password
  • event - event when log-in button is clicked
forget-password: & function(event) Callback function when click forget password.
  • event - event when click forget-password
on-sign-up: & function(event) Callback function when sign-up button is clicked.
  • event - event when sign-up button is clicked
