Overview

Angular version of validation component which is implemented by using .component() method introduced since Angular1.5.

wf-validation settings

Attributes:Bindings Accept value Note
pattern: <
  • ip_address
  • date
  • dateWP
  • time
 The type of validations.
regex: <
  • regex
 Regex is used to transfer userDefined regex.When either of the two parameters(Regex and pattern) are both not undefined.Regex has priority over pattern.
