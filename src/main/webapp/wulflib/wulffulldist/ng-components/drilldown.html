Overview

Angular version of Drilldown component which is implemented by using .component() method introduced since Angular1.5. Involve JS 'wulf-ng/drilldown/drilldown' and add the module angular.module('myApp',['wf.angular.drilldown'] to your angular application. Add class n-drillDown-item and set data-target-selector="#drilldown component identity" in td. Put wf-drilldown in next tr which will drill down as click. Now drill down component can be worked.

wf-drilldown settings

Attributes:Bindings Accept value Note
identity: @ String The identity text which unique define the component
content: & function() Function should return the drilldown content, the content can be text or other html component.
arrow: <?
  • true
  • false (Default)
 Indicate whether arrow show or not when content expanded.
height: @ String(Optional, default is 100px). The content height which will drill down. e.g 200px

Keyboard accessibility

Besides the basic keyboard support, e.g. Tab, Shift + Tab and Enter. This component also supports the following keystrokes for the interaction.
Interaction Keystrokes Notes
Expand/Collapse Spacebar/Enter - Expand or Collapse the content
