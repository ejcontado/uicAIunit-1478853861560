Tab buttons



Overview

Angular version of tab buttons component. It contains a component wfTabButton which are implemented by using .component() method introduced since Angular1.5. This component can be used to generate tab buttons from dynamic model data.

wf-tab-button settings

Attributes:Bindings Default/Arguments Notes
button-info: < array (Mandatory) The tab buttons config object. The object indicate the information of tab buttons. These properties are currently supported:
  • text The displayed text in button.
  • disable indicate whether the button is disable (Default: false).
For example: [{text: "Button1", disable: false}, {text: "Button2", disable: false}, {text: "Button3", disable: false}]
disable: <?
  • true
  • false (Default)
 Whether buttons is clickable and able to be activated
pattern: @
  • standard (Default)
  • small
 The pattern of buttons.
on-click-callback: & function (event) Callback function for click event. It will be called after you clicked the item from tab buttons.

Keyboard accessibility

Besides the basic keyboard support, e.g. Tab, Shift + Tab and Enter. This component also supports the following keystrokes for the interaction.
Interaction Keystrokes Notes
Move focus to next/previous item ← or → - previous/next item
  • The disabled item would be skipped
  • Using Tab or Shift + Tab can navigate out of the group of items
Select item Spacebar/Enter - Select current focused item
  • The disabled item cannot be selected
 
                    



    
    

    
    



                    

angular.module('wf.ng.demo', ['wf.angular.actionbuttons','wf.angular.tabs']).controller('tabButtonsDemoController', tabButtonsDemoController);
function tabButtonsDemoController(){
    var newInstance = function (buttonInfo, disable) {
        return {
            buttonInfo: buttonInfo,
            disable: disable
        }
    };
    var self = this;
    self.tabButtonsData = newInstance([{text: "Button1", disable: false}, {text: "Button2", disable: false},
        {text: "Button3", disable: false}], false);
    self.buttonClickedCallback = function (event) {
        console.log(event.target);
    };
}
angular.element(document).ready(function () {
    angular.bootstrap(document, ['wf.ng.demo'], {
        strictDi: true
    });
});
