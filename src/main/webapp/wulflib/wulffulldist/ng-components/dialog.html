Overview

Angular version of dialog component which is implemented by using .component() method introduced since Angular1.5.

wf-dialog settings

Attributes:Bindings Accept value Note
pattern: <
  • standard (Default)
  • error
  • warning
  • confirmation
 The pattern of dialog to be created.
title-text: < String (Default: null) The title text of dialog.
subheader: < String (Default: null) The sub header text of dialog.
instruction: < String (Default: null) The instruction text of dialog.
active: <?
  • false (Default)
  • true
 Active the dialog or not.
confirm-btn-text: < String (Default: OK) The text in confirm button of dialog.
toggle-style: <
  • link (Default)
  • button
 The toggle style of dialog. Can be link or button
toggle-text: < String (Default: null) The toggle text of dialog.
animation: <?
  • false (Default)
  • true
 Enable the animation or not.
on-confirm: & function(event) Callback function when confirm button is clicked.
  • event - event when confirm button is clicked
on-cancel: & function(event) Callback function when cancel button is clicked.
  • event - event when cancel button is clicked
on-toggle: & function(isShow, event) Callback function when dialog is toggled.
  • isShow - Whether the dialog is show or hide
  • event - event when toggle dialog

Keyboard accessibility

Besides the basic keyboard support, e.g. Tab, Shift + Tab and Enter. This component also supports the following keystrokes for the interaction.
Interaction Keystrokes Notes

    angular.module('wf.ng.demo', ['wf.angular.dialog', 'wf.angular.tabs']).controller('DialogDemoController', DialogDemoController);
    function DialogDemoController() {
        this.pattern = 'standard';
        this.title = 'Standard Dialog';
        this.subheader = 'Subheader';
        this.instruction = 'Instruction';
        this.isActive = true;
        this.btnText = 'OK';
        this.toggleStyle = 'link';
        this.toggleText = 'Standard Dialog';
        this.animation = true;
        this.onConfirm = function() {
            alert('Confirm button is clicked');
        };
        this.onCancel = function() {
            alert('Cancel button is clicked');
        };
        this.onToggle = function(isShow){
            var status = isShow? 'shown':'hidden';
            alert('Dialog will be ' + status);
        }
    }
