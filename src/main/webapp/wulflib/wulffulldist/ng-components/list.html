Overview

Angular version of lists. It contains two components. wf-list which represents the container list and wf-list-item which represents a list item.

wf-list settings

Attributes:Bindings Accepted values Description
type: <
  • basic
  • selection
 The type of list. Value selection allows list items to be selected via click or keyboard shortcuts.
disable: <?
  • false (Default)
  • true
 If true, disables the entire list along with the list items. Note that the selected item will still be selected when disabled/re-enabled.
design: @
  • standard (Default)
  • striped
  • dark
 Design of the entire list
heading: @
  • Any string
 Intended heading for the list
on-item-selected: & function(index, heading, event) Callback function when list item is selected.
  • index - current item index of the list
  • heading - heading of the current list
  • event - event when item is selected

wf-list-item settings

Attributes:Bindings Accepted values Description
select: <?
  • true
  • false (default)
 If true, sets the list item to be selected. This attribute will have no effect if list is not of type selection. Selected item persists on toggling disable. End users can easily override this behavior by setting select to false.

Keyboard accessibility

Interaction Keystrokes Notes
Moves the list item focus ↓ / ↑ Focus normally spawns focus from the currently selected item.
Selects the currently focused item Space / Enter Will do nothing if nothing is focused

Missing functionality

  • Scrollbar not support yet.

angular.module('wf.ng.demo', ['wf.angular.list', 'wf.angular.tabs'])
.controller('DemoController', function () {
    this.plainList = {};
    this.selectionList = {};
    this.stripedList = {};
    this.darkList = {};

    this.demoItems = [
        { text: "Item one" },
        { text: "Item two" },
        { text: "Item three" }
    ];

    this.demoItemsWithSelectedText = [
        { text : "Item one" },
        { text : "This item should be selected...", select: true },
        { text : "This is a very very very very very very very very very very very very very very long text, so much so that it should wrap to the next line." }
    ];

    this.plainList.items = this.demoItems;
    this.plainList.disabled = false;

    this.selectionList.items = this.demoItemsWithSelectedText;
    this.selectionList.disabled = false;

    this.stripedList.disabled = false;
    this.darkList.disabled = false;

    this.onItemSelected = function (index, heading) {
        alert(heading +'  item '+ index +' is selected');
    };
});
