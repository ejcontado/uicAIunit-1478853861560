Spinner


Overview

Angular version of spinner component. It is implemented by using .component() method introduced since Angular1.5. This component can be used to present and edit a value from within a discrete range of values.

wf-spinner settings

Attributes:Bindings Accept value Note
min: < Number (Default: 0) The minimum value of the spinner.
max: < Number (Default: 999) The maximal value of the spinner.
init: < Number (Optional) The initial value of the spinner. The default initial value is minimum value.
step: < Number (Default: 1) The step value when increase or decrease.
pattern: @
  • standard(Default)
  • small
 The type of spinner.
disable: <?
  • true
  • false(Default)
 The status of spinner.
on-value-changed: & function (currentValue, event) Callback function when spinner value changed.
  • currentValue - changed value of spinner
  • event - event when spinner's value is changed

Keyboard accessibility

Besides the basic keyboard support, e.g. Tab, Shift + Tab and Enter. This component supports the following keystrokes for the interaction.
Interaction Keystrokes Notes
Increase/decrease spinner value ↓ or ↑ Only in the inputfield
Trigger increase/decrease button Spacebar/Enter 
                    
    

            
            
    

                    
                
    
    angular.module('wf.ng.demo', ['wf.angular.spinner']).controller('spinnerDemoController', spinnerDemoController);
    function spinnerDemoController() {
        var newInstance = function (min, max, init, step, disable) {
            return {
                min: min,
                max: max,
                init: init,
                step: step,
                disable: disable
            }
        };
        var self = this;
        var onValueChanged = function (currentValue) {
            if (self.actionState) {
                alert('current value will change to ' + currentValue);
            }
        };
        self.spinner1 = newInstance(0, 999, 1, 1, false);
        self.spinner1.onValueChanged = onValueChanged;
        self.spinner2 = newInstance(0, 999, 1, 1, false);
        self.spinner2.onValueChanged = onValueChanged;
    }
    angular.element(document).ready(function () {
        angular.bootstrap(document, ['wf.ng.demo'], {
            strictDi: true
        });
    });
