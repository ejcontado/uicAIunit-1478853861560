Search

Overview

Angular version of Search component. This component can be used to provide search entrance to user

Settings

Attributes:Bindings Accept value Note
placeholder: @ text Placeholder to show.
type: @
  • 'standard' (Default)
  • 'clear'
  • 'dropDown'
  • 'clearDropDown'
  • 'searchBtn'
  • 'searchBtnDropDown'
 To define which type of search component you want to use.
search-btn-text: @ text Text on search btn to show.
disable: <
  • true
  • false (Default)
 Whether it's disabled.
on-submit-callback: & function (event,textValue) Callback function defined by parent. It would be called after you clicked the search button on the right.
drop-down-data: <
  • data array like : [{text:'abc',value:123}]
 The key of your data item must follow this introduction.
on-item-selected-callback: & function (event,itemData,textValue) Callback function from parent controller. It would be called after you clicked the item from the drop down list.

Keyboard accessibility

Besides the basic keyboard support, e.g. Tab, Shift + Tab and Enter. This component also supports the following keystrokes for the interaction.
Interaction Keystrokes Notes
Move focus to next/previous item ↑ or ↓ - previous/next item
  • The disabled item would be skipped
  • Using Tab or Shift + Tab can navigate out of the group of items
Select item Spacebar/Enter - Select current focused item
  • The disabled item cannot be selected
Trigger clear button Spacebar/Enter - Click on clear button 
                    











                    
                
                    
        //this is a user scenario
       var demo = angular.module('wf.ng.demo', ['wf.angular.search', 'wf.angular.tabs']);
        var demoController = function () {
            var thiz = this;
            thiz.dropDownData = [{text:'savedSearch1',value:'aaa'},{text:'savedSearch2',value:'bbb'},{text:'savedSearch3',value:'ccc'}];
            thiz.searchBtnClickedCallback = function (event,textValue) { //user can defined his own callback using this parameters
                console.log('input text is : '+ textValue);
            }
            thiz.textValue2 = '' ;
            thiz.textValue1 = '' ;
            thiz.textValue0 = '' ;
            thiz.demoItemSelect = function (event,selectedItem,textValue){ //user can defined his own callback using this parameters
                var tagType = angular.element(event.target).parents('wf-search').attr ('type');
                if(tagType === 'searchBtnDropDown'){
                    thiz.textValue2  = selectedItem.text;
                }else
                if(tagType === 'clearDropDown'){
                    thiz.textValue1  = selectedItem.text;
                }else
                if(tagType === 'dropDown'){
                    thiz.textValue0  = selectedItem.text;
                }
                console.log('selected item data is '+selectedItem.toString() + ",and textValue is : "+ textValue);
            }
            thiz.btnText = 'Disable All';
            thiz.disable=false;
            thiz.disableBtnClicked = function () {
                thiz.disable = !thiz.disable;
                thiz.btnText = thiz.btnText=='Disable All'?'Enable All':'Disable All';
            }
        };
        demo.controller('DemoController', demoController);
