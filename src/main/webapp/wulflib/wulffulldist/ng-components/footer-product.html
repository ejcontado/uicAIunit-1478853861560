Footer

Overview

Angular version of Footer component which is implemented by using .component() method introduced since Angular1.5. The footer is used to display some links and copyright.This component can be generated footer in the bottom of the page.

wf-footer settings

Attributes:Bindings Accept value Note
copyright: < String Copyright text
links: < JS array object Object structure: [{name: "Link1", href: "#"}]
icons: < JS array object Object structure: [{href: "#", src: "../assets/images/wulf.png"}], href is optional.
