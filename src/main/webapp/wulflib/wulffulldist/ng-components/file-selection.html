File Selection




Overview

Angular version of tabs component. It contains two components wfCombobox and wfComboboxItem which are implemented by using .component() method introduced since Angular1.5. This component is used to upload the file and display the file name.

wf-combobox settings

Attributes:Bindings Accept value Note
pattern: < file-selection The pattern of combo box to be created.
It is used to match suitable template for file upload component.
size-style: <
  • standard(Default)
  • small
 Choose the size of the file selection component

wf-combobox-item settings

Attributes:Bindings Accept value Note
item-value: < String (Default is Browse...) The value for combo box item.
get-file: & function (fileName, filePath) Callback function when select file.
  • fileName - the name of the file that you have chosen
  • filePath - the path of the file storage

Keyboard accessibility

Besides the basic keyboard support, e.g. Tab, Shift + Tab and Enter. This component also supports the following keystrokes for the interaction.
Interaction Keystrokes Notes
Move focus to next/previous item ↑ or ↓ - previous/next item
  • Using Tab or Shift + Tab can navigate out of the group of items
Select item Spacebar/Enter - Select current focused item

Missing functionality

  • Scrollbar not support yet.
  • Tooltip not support yet.
                    
    

          
       
           
       

        
        
            
        

    

                    
                
                    
    angular.module('wf.ng.demo', ['wf.angular.combobox', 'wf.angular.tabs']).controller('FileSelectionDemoController', FileSelectionDemoController);

    function FileSelectionDemoController (){
                var self = this;
                this.fileSelectionCombobox = {};
                this.smallFileSelection = {};
                this.itemValue;
                this.pattern = 'file-selection';
                this.fileSelectionCombobox.sizeStyle = 'standard';
                this.smallFileSelection.sizeStyle = 'small';
                this.getFile = function (fileName,filePath) {
                    if (self.actionState){
                        alert("\nFileName: " + fileName  + "\nFilePath: " + filePath);
                    }
                };
                this.actionState = false;
                this.enableDisableAction = function () {
                    this.actionState = !this.actionState;
                };
            }
