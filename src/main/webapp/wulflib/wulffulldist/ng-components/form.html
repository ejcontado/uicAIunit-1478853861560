Form

Overview

We did not capsulate form but just show how to use directive to update component status. Detail angular input text information please refer to https://docs.angularjs.org/api/ng/directive/form.
Angular provides properties on forms that help us validate them. They give us various information about a form or its inputs and are applied to a form and inputs.

Property Class Description
$valid ng-valid Boolean Tells whether an item is currently valid based on the rules you placed
$invalid ng-invalid Boolean Tells whether an item is currently invalid based on the rules you placed.
$invalid ng-invalid Boolean Tells whether an item is currently invalid based on the rules you placed.
$pristine ng-pristine Boolean True if the form/input has not been used yet.
$dirty ng-dirty Boolean True if the form/input has been used.

Angular also provides classes on the form and its inputs so that you can style each state accordingly.

A few key points to note here:
novalidate: This will prevent the default HTML5 validations
We have applied ng-model to our inputs so that we have data from our forms bound to Angular variables.
If password is require, add required to the input element. The email input is type="email"

Validation Rules Angular provides many validation rules that we can use in addition to ng-minlength and ng-maxlength.
These are the available parameters for an Angular input to create validation rules. Read the Angular input directive for more information.
<input ng-model="{string}" name="{string}" required ng-required="{boolean}" ng-minlength="{number}" ng-maxlength="{number}" ng-pattern="{string}" ng-change="{string}"></input>

Disabling the Submit Button ng-disabled:
<button type="submit" class="btn" ng-disabled="wfForm.$invalid">Submit</button>

With just that little code (ng-disabled), our form button will be disabled if the form is $invalid. This means that our password input field is required and our email input field requires a valid email.

Showing an Error Message ng-if:
ng-valid and ng-invalid will automatically determine if an input is good based on the rules placed on it in your form.
Angular will automatically determine if we should show an error based on an inputs $invalid and $pristine properties.
ng-class allows us to add classes based on an expression. In this case, we want to add a input-error class to our form-group if an input is $invalid and not pristine.
The way it works is ng-class="{ <class-you-want> : <expression to be evaluated > }". For more information, read the Angular ngClass docs.

Submitting a form:
You can use one of the following two ways to specify what javascript method should be called when a form is submitted:
ngSubmit directive on the form element
ngClick directive on the first button or input field of type submit (input[type=submit])
To prevent double execution of the handler, use only one of the ngSubmit or ngClick directives. This is because of the following form submission rules in the HTML specification:

require(['angular', 'angularAnimate', 'wulf-ng/panels/panels', 'wulf-ng/dropdowns/dropdowns',
'wulf-ng/form/form','wulf-ng/tabs/tabs'], function(angular) {
    angular.module('wf.ng.demo', ['wf.angular.panels', 'wf.angular.dropdowns',
    'wf.angular.form','wf.angular.tabs']).controller('formDemoController', formDemoController);

    function formDemoController() {
        var makeSection = function(title, divClass, content) {
            return {
                title: title,
                divClass: divClass,
                content: content
            }
    };

    this.disable = false;

    var newPullDownInstance = function(type, disable) {
        return {
            type: type,
            disable: disable
        }
    };

    var newPullDownItemInstance = function(value, label, disable, select) {
        return {
            value: value,
            label: label,
            disable: disable,
            select: select
        }
    };

    this.standardPullDownInstance = newPullDownInstance("standard", false);
    this.items = [newPullDownItemInstance('female', 'female', false, true),
    newPullDownItemInstance('male', 'male', false, false)];}

    angular.element(document).ready(function() {
        angular.bootstrap(document, ['wf.ng.demo'], {
            strictDi: true
        });
    });
});
