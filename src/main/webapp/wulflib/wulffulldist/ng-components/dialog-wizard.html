Overview

Angular version of dialog wizard component which is implemented by using .component() method introduced since Angular 1.5. This inherits and extends WULF's Angular Dialog functionality.

wf-dialog settings (Wizard specific)

Attributes:Bindings Accept value Note
is-wizard: <?
  • true
  • false (default)
 If true, structures the dialog to generate Step information and Next/Previous buttons for step navigation
wf-wizard does not have binding settings, it is only meant to mark html and impose structure.

wf-wizard-step settings

Attributes:Bindings Accept value Note
bulletText: < Any short character Any relevant character. Be wary of the text size since it is only enclosed by a small circle.
setActive: <?
  • true
  • false (default)
 If true, sets the step as the active step and assumes the previous steps to be done. There must be ONLY ONE step set as active at one time.
on-step-traversed: & Callback function Called when the step is traversed, either by the Next button or the Previous button

wf-wizard-step transclude slots

Slot name Note
wf-wizard-step-description General description of the step, accepts any html markup including Angular.
wf-wizard-step-content Specific content of the step, accepts any html markup including Angular.

Keyboard accessibility

Shares keyboard functionality with base WULF Angular Dialog.

require(['angular', 'wulf-ng/dialog/dialog'], function (angular) {
    angular.module('wf.ng.demo', ['wf.angular.dialog']).controller('DialogDemoController', DialogDemoController);

    function DialogDemoController() {
        this.animation = false;

        this.wizard = {};
        this.wizard2 = {};

        this.wizard.title = "Create Task";
        this.wizard.subheader = "General data";
        this.wizard.isActive = true;
        this.wizard.btnText = "Finish";
        this.wizard.toggleStyle = "link";
        this.wizard.pattern = "standard";
        this.wizard.toggleText = 'Wizard Dialog';

        this.wizard.stepDetails = [
            {
                description: 'Description Step 1',
                content: 'Content Step 1',
                active: true,
                bulletText: 'I'
            },
            {
                description: 'Description Step 2',
                content: 'Content Step 2',
                bulletText: 'II'
            },
            {
                description: 'Description Step 3',
                content: 'Content Step 3',
                bulletText: 'III'
            }
        ];


        this.wizard2.toggleStyle = "button";
        this.wizard2.toggleText = 'Registration Wizard Dialog';
        this.wizard2.title = "Registration";
    });
