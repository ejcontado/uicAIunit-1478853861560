Timezone

Overview

Angular version of timezone component. It contains a component wfTimezone which are implemented by using .component() method introduced since Angular1.5. This component can be used to generate timezone from dynamic model data.

wf-timezone settings

Attributes:Bindings Default/Arguments Notes
default-zone: < string The default selected timezone to show. Example: 'Europe/Helsinki'
zone-data-bundle: < array (Optional) Load a custom data bundle of timezones. These properties are currently needed:
  • name Timezone name.
  • offset Timezone offset.
Example: [{name: "Africa/Abidjan", offset: "+00:00"}, {name: "Africa/Accra", offset: "+00:00"}].
If there is no parameter transferred, this component will display as "Normal" in left.
disable: <?
  • true
  • false (Default)
 Whether timezone is clickable and able to be activated
on-item-selected: & function (event) Callback function for item click event. It will be called after you clicked the item in pulldown options.

Keyboard accessibility

Besides the basic keyboard support, e.g. Tab, Shift + Tab and Enter. This component also supports the following keystrokes for the interaction.
Interaction Keystrokes Notes
Move focus to next/previous item ↑ or ↓ - previous/next item
  • The disabled item would be skipped
Select item Spacebar/Enter - Select current focused item
  • The disabled item cannot be selected
 
                    



    
    

    
    



                    

angular.module('wf.ng.demo', ['wf.angular.timezone','wf.angular.tabs']).controller('timezoneDemoController', timezoneDemoController);
function timezoneDemoController(){
    var newInstance = function (dataBundle, defaultZone, disable) {
        return {
            dataBundle: dataBundle,
            defaultZone: defaultZone,
            disable: disable
        }
    };
    var self = this;
    var dataBundle = [
        {name: "Africa/Abidjan", offset: "+00:00"},
        {name: "Africa/Accra", offset: "+00:00"},
        {name: "Africa/Addis_Ababa", offset: "+03:00"},
        {name: "Pacific/Niue", offset: "-11:00"},
        {name: "Pacific/Pago_Pago", offset: "-11:00"},
        {name: "Africa/Asmera", offset: "+03:00"},
        {name: "Africa/Bamako", offset: "+00:00"},
        {name: "Africa/Bangui", offset: "+01:00"},
        {name: "Africa/Banjul", offset: "+00:00"},
        {name: "Africa/Bissau", offset: "+00:00"}
    ];
    self.timezoneData1 = newInstance(undefined, 'Europe/Helsinki', false);
    self.timezoneData1.itemSelectedCallback = function (event) {
        alert(event.currentTarget.innerText);
    };
    self.timezoneData2 = newInstance(dataBundle, 'Africa/Asmera', false);
    self.timezoneData2.itemSelectedCallback = function (event) {
        alert(event.currentTarget.innerText);
    };
};
angular.element(document).ready(function () {
    angular.bootstrap(document, ['wf.ng.demo'], {
        strictDi: true
    });
});
