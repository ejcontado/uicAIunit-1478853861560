Table

Angular version of table(Smart table). It allows you to quickly compose your table in a declarative way including sorting, filtering, row selection, pagination and indicators.
The first thing when enable smart table is to require dependency js 'wulf-ng/tables/smart-table' and add the module angular.module('myApp',['wf.angular.tables'] to your angular application. Then use the markup you would normally do for html tables. On the table element add the wf-table attribute to tell smart-table which collection holds your displayed data, (ie the one you are going to use in the repeater). Now you will be able to compose the table using the plugins.

Settings

Directive attribute Accept value Note
wf-table JS object(Array) The displayed data in the table
wf-sort The key in JS object We can use the wf-sort directive to have a built in sort feature. Again, it is all declarative. You add the attribute wf-sort to your table headers. This will specify the property name you want to sort by or the name of a getter function available on the scope. There are three different sort states: ascending, descending and back to the primary order
wf-skip-natural
  • true
  • false (Default)
 Skip the "natural order" state by adding wf-skip-natural="true" as attribute of your th element
wf-search The key in JS object The wf-search gives you a way to filter rows globally if you don't set any value to the attribute. Or you can also define the predicate to use as value of the directive attribute. Use a property name, or alternatively you can interpolate the predicate with a scope property
wf-clear NA Clear all the condition values in search function. Binding element click event to reset
wf-pagination NA Enable pagination function and pagination template will be displayed. Pagination template includes summary information and paging button
wf-items-by-pag Numberic The item number in one page
wf-summary NA Enable calculate summary records function. Summary information includes total records and current records
wf-indicator JS object Watch the value in object, show indicator when value is changed

Keyboard accessibility

Smart table support the basic keyboard action, e.g. Tab, Shift + Tab, ↑, ↓, ← and →.

Note

WULF smart table use for reference of angular smart table(http://lorenzofox3.github.io/smart-table-website/). WULF smart table covers all the features of angular smart table and make some templates for WULF style.
For more detail directive information, please refer to http://lorenzofox3.github.io/smart-table-website/.
