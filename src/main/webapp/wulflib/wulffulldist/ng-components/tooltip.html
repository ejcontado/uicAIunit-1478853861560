Tooltips can be used to display quick information on an element in the page. To use tooltip in Angular in WULF, add the directive wf-tooltip.

wf-tooltip settings

  • wf-context: '@' - This will determine the content of your tooltip
  • wf-location: '=' - This will define the location of the tooltip in accordance to its parent component It can be set to left, right, top or bottom 

<a wf-tooltip wf-location="top" wf-context="Tooltip Top" href="javascript:void(0)">top </a>
