Nokia WULF is a HTML, CSS and JS framework for developing web applications

  • Follows Nokia Look & Feel Guidelines - Ocean Touch 3.0 (OT3.0) and Ocean Touch 3.1 (OT3.1)
  • Easy to update for future L&F rules, easily upgradeable platform solutions
  • Responsive, scaling to different screen sizes from phones to big PC screens
  • Light-weight and small, not using pictures
  • Accessibility taken into account
  • Cross-browser support
  • Multiple variants for different technologies

DRAFT statement in component means that it is not yet ready (under construction still). It is possible for you to use such component, but please note that it can change still - a lot even.
If the draft statement appears in the component header, that whole component is draft. If there are draft statements inside the component, then only those sub-parts are draft.
